## 四、Markdown格式规范

### 4.1 标题层级

- 一级标题：文档标题（仅一个）
- 二级标题：章节
- 三级标题：小节
- 四级标题：子小节

**不超过四级标题**。

### 4.2 代码块

指定语言类型：

````markdown
```python
# Python代码
def hello():
    print("Hello, World!")
```

```bash
# Shell命令
python main.py
```

```yaml
# YAML配置
key: value
```
````

### 4.3 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容 | 内容 | 内容 |
```

### 4.4 列表

无序列表：
```markdown
- 项目1
- 项目2
  - 子项目2.1
  - 子项目2.2
```

有序列表：
```markdown
1. 步骤1
2. 步骤2
   1. 子步骤2.1
   2. 子步骤2.2
```

### 4.5 链接和图片

```markdown
[链接文字](URL)

![图片说明](图片路径)
```
