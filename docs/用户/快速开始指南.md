## 一、欢迎来到智能小说创作助手

### 1.1 这是什么？

Novel Writing Assistant-Agent Pro 是一款基于AI的智能小说创作辅助工具，帮助你从零开始构建完整的小说世界，包括世界观设定、人物塑造、大纲规划、风格学习和章节生成。

### 1.2 核心功能一览

| 功能 | 描述 | 适用场景 |
|------|------|----------|
| **世界观解析** | 从已有文本提取世界观设定 | 学习优秀作品的设定技巧 |
| **人物管理** | 创建和管理立体丰满的人物形象 | 角色设计、人设维护 |
| **大纲生成** | 自动生成章节规划和剧情走向 | 项目规划、剧情设计 |
| **风格学习** | 学习特定作者的写作风格 | 风格模仿、文笔提升 |
| **智能创作** | AI辅助章节内容生成 | 快速成稿、突破卡文 |
| **快捷创作** | 一键生成世界观/大纲/人物/情节 | 新项目启动、设定扩展 |
| **续写功能** | 智能续写已有章节内容 | 灵感延续、剧情延伸 |
| **逆向反馈** | 分析已有文本并生成创作建议 | 作品优化、问题诊断 |
| **热榜功能** | 查看当前热门题材和作家趋势 | 市场分析、选题参考 |

## 二、安装与配置

### 2.1 系统要求

- **操作系统**: Windows 10/11（推荐）或 Windows 7+
- **Python**: 3.12.x（必须）
- **内存**: 8GB+（推荐16GB）
- **硬盘**: 5GB可用空间（含模型缓存）

### 2.2 安装步骤

#### 方法一：使用一键启动脚本（推荐）

1. **解压项目文件**到任意目录
2. **双击运行** `一键启动.bat`
3. 等待依赖自动安装（首次运行约5-10分钟）
4. 看到主界面即表示启动成功

#### 方法二：手动安装

```bash
# 1. 进入项目目录
cd "Novel Writing Assistant-Agent Pro"

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置API密钥
# 复制配置模板
copy config.yaml.example config.yaml

# 编辑config.yaml，填入你的API密钥
# 推荐使用DeepSeek API（性价比最高）

# 4. 启动程序
python gui_main.py
```

### 2.3 API密钥配置

**重要**：必须配置API密钥才能使用AI功能。

#### 推荐方案：DeepSeek API

1. 访问 [DeepSeek官网](https://www.deepseek.com/) 注册账号
2. 进入API控制台获取API Key
3. 编辑 `config.yaml`：

```yaml
deepseek:
  model: deepseek-chat
  api_key: 你的API密钥
  base_url: https://api.deepseek.com
```

**费用参考**（2026年3月）：
- 输入：¥0.14/百万token
- 输出：¥0.28/百万token
- 相比OpenAI便宜约70倍

#### 支持的其他API

- **OpenAI**: 兼容GPT-4、GPT-3.5
- **Anthropic**: 兼容Claude系列
- **本地Ollama**: 支持本地部署模型

**侧边导航菜单**：

- 🔌 插件管理器 - 查看和管理已安装插件
- ⚙️ 设置 - 程序配置和API设置

### 3.2 创建第一个项目

#### 步骤1：进入项目管理

点击侧边栏的 **📁 项目管理**，然后点击 **新建项目**。

#### 步骤2：填写项目信息

```
项目名称：我的修仙小说
题材类型：玄幻
目标字数：100万
```

点击 **创建**，项目文件夹将自动创建。

#### 步骤3：生成基础设定

进入 **📝 工作台 → 快捷创作**：

1. **输入关键词**：
   ```
   修仙、逆袭、系统、复仇
   ```

2. **选择生成类型**：
   - 勾选：世界观、大纲、人物、情节
   - 详细程度：标准
   - 题材：玄幻

3. **点击生成**，等待2-3分钟

4. **预览结果**：
   - 世界观：修仙境界体系、宗门势力分布
   - 大纲：50章剧情规划
   - 人物：主角+核心配角
   - 情节：开篇、转折、高潮设计

5. **保存到项目**：点击 **导入当前项目**

### 3.3 生成第一章节

#### 方法一：智能创作

1. 进入 **工作台 → 创作**
2. 选择章节：第一章
3. 设置参数：
   ```
   字数要求：3000字
   风格参考：流畅、生动、引人入胜
   ```
4. 点击 **开始生成**
5. AI将基于世界观、人物、大纲自动生成章节内容

#### 方法二：续写功能

如果你已经写了开头，可以使用续写功能：

1. 进入 **工作台 → 续写**
2. 输入已有内容或导入文本文件
3. 设置续写参数：
   ```
   续写字数：2000字
   剧情走向：保持原风格，推进主线
   ```
4. 点击 **开始续写**

## 四、核心工作流程

### 4.1 从零开始创作（推荐新手）

```
1. 创建项目 → 2. 快捷创作生成设定 → 3. 优化调整设定
                ↓
4. 规划章节大纲 → 5. 风格学习（可选）→ 6. 开始章节创作
                ↓
7. 质量验证 → 8. 迭代优化 → 9. 完成发布

                ↓
7. 智能创作 → 8. 质量验证 → 9. 完成发布
```

## 五、常用技巧

### 5.1 提高生成质量的技巧

#### 技巧1：详细的关键词

❌ 差的关键词：
```
修仙
```

✅ 好的关键词：
```
修仙、宗门争霸、天赋觉醒、废柴逆袭、师徒情深、飞升成仙
```

#### 技巧2：使用参考文本

在快捷创作中，可以导入你喜欢的小说片段作为参考，AI会学习其设定风格。

#### 技巧3：风格学习

如果你想模仿特定作者的风格：

1. 进入 **工作台 → 风格**
2. 导入该作者的作品（至少10章）
3. 点击 **学习风格**
4. 生成时选择使用该风格

### 5.2 项目管理技巧

#### 技巧1：合理分类

建议在项目中创建子文件夹：
```
我的修仙小说/
├── 世界观/
├── 人物设定/
├── 章节大纲/
├── 正文/
├── 参考资料/
└── 灵感笔记/
```

#### 技巧2：定期备份

定期将项目文件夹复制到其他位置或云盘，避免数据丢失。

#### 技巧3：版本控制

对重要章节建议保存多个版本：
```
第1章_v1_初稿.txt
第1章_v2_修改稿.txt
第1章_v3_定稿.txt
```

### 5.3 节省API费用的技巧

#### 技巧1：使用详细程度控制

- **快速生成**：适合初期探索，消耗token少
- **标准生成**：平衡质量和成本
- **详细生成**：最终定稿时使用

#### 技巧2：分步骤生成

不要一次性生成所有内容，而是：
1. 先生成世界观框架
2. 满意后再生成大纲
3. 最后生成章节

这样可以避免不满意时重新生成造成的浪费。

#### 技巧3：复用已有设定

已生成的世界观、人物设定可以在多个项目中复用，避免重复生成。
