## 一、长期记忆系统

### 1.1 OpenClaw mem9架构概述

OpenClaw mem9 是一套五层记忆架构,旨在为AI创作助手提供长期记忆能力,确保长篇创作的连贯性和一致性。

#### 1.1.1 架构层次

**L1热记忆 - SESSION-STATE**
- 存储位置:内存
- 生命周期:会话期间
- 内容:当前会话状态、活跃人物、当前场景
- 访问速度:毫秒级
- 容量:有限(受内存限制)

**WAL协议 - Write-Ahead Logging**
- 作用:原子化写入保证
- 机制:先写日志,后执行操作
- 保证:数据一致性和可恢复性
- 实现:core/wal_manager.py

**L2温记忆 - LanceDB向量库**
- 存储位置:磁盘(向量数据库)
- 生命周期:持久化
- 内容:章节向量、人物向量、事件向量
- 访问速度:10-100毫秒
- 容量:大规模(百万级向量)

**L3冷记忆 - Git-Notes**
- 存储位置:Git仓库
- 生命周期:永久
- 内容:历史决策、重要变更、创作里程碑
- 访问速度:秒级
- 容量:无限制

**L4档案 - MEMORY.md**
- 存储位置:项目目录
- 生命周期:永久
- 内容:精炼的知识、最佳实践、重要经验
- 访问速度:毫秒级(文本文件)
- 容量:适中(MB级)

#### 1.1.2 数据流转

```
用户输入
    ↓
L1热记忆处理(实时)
    ↓
WAL协议保证(原子化)
    ↓
L2温记忆索引(向量检索)
    ↓
L3冷记忆归档(Git-Notes)
    ↓
L4档案沉淀(MEMORY.md)
```

### 1.3 WAL协议详解

#### 1.3.1 WAL工作原理

**写入流程**:
```
1. 接收写入请求
2. 序列化操作数据
3. 写入WAL日志文件
4. 刷新到磁盘(fsync)
5. 执行实际操作
6. 清理WAL日志
```

**恢复流程**:
```
1. 系统启动
2. 读取WAL日志文件
3. 重放未完成的操作
4. 恢复到一致状态
5. 清理WAL日志
```

#### 1.3.2 WAL配置

**配置参数**:
```yaml
wal:
  enabled: true
  log_dir: ./data/wal/
  max_log_size: 10MB  # 单个日志文件最大大小
  max_log_count: 5    # 保留日志文件数量
  sync_interval: 100  # 同步间隔(毫秒)
```

#### 1.3.3 WAL监控

**监控指标**:
- 写入延迟:平均写入时间
- 日志大小:当前日志文件大小
- 队列长度:待写入操作队列长度
- 错误率:写入失败率

**告警规则**:
- 写入延迟 > 100ms:性能警告
- 日志大小 > 50MB:容量警告
- 错误率 > 1%:错误警告

### 1.4 L2温记忆使用

#### 1.4.1 向量检索原理

**向量编码**:
- 模型:sentence-transformers/all-MiniLM-L6-v2
- 维度:384维
- 类型:密集向量
- 支持:中英文

**相似度计算**:
- 算法:余弦相似度
- 公式:cos(A,B) = (A·B) / (||A|| × ||B||)
- 范围:-1到1(1表示完全相同)

**检索策略**:
- Top-K检索:返回相似度最高的K个结果
- 阈值过滤:只返回相似度高于阈值的结果
- 混合检索:向量检索+关键词检索

### 3.1 Agent系统架构

#### 3.1.1 MasterAgent总控

**职责**:

- 结果整合
- 异常处理

**调度算法**:
```python
class MasterAgent:
    def execute(self, task):
        # 1. 分析任务
        subtasks = self.decompose(task)
        
        # 2. 选择执行策略
        strategy = self.select_strategy(subtasks)
        
        # 3. 分配给专家Agent
        results = []
        for subtask in subtasks:
            agent = self.select_agent(subtask)
            result = agent.execute(subtask)
            results.append(result)
        
        # 4. 整合结果
        final_result = self.integrate(results)
        
        return final_result
```

### 3.3 自定义Agent开发

#### 3.3.1 Agent基础类

```python
from abc import ABC, abstractmethod
from typing import Any, Dict

class BaseAgent(ABC):
    """Agent基类"""
    
    def __init__(self, name: str, config: Dict[str, Any]):
        self.name = name
        self.config = config
    
    @abstractmethod
    def execute(self, task: Dict[str, Any]) -> Dict[str, Any]:
        """执行任务"""
        pass
    
    def validate_task(self, task: Dict[str, Any]) -> bool:
        """验证任务"""
        required_fields = self.get_required_fields()
        return all(field in task for field in required_fields)
    
    @abstractmethod
    def get_required_fields(self) -> list:
        """获取必需字段"""
        pass
```

#### 3.3.2 实现自定义Agent

```python
from agents.base_agent import BaseAgent

class CustomAnalyzerAgent(BaseAgent):
    """自定义分析Agent"""
    
    def __init__(self, config: Dict[str, Any]):
        super().__init__("CustomAnalyzer", config)
    
    def execute(self, task: Dict[str, Any]) -> Dict[str, Any]:
        # 验证任务
        if not self.validate_task(task):
            raise ValueError("任务缺少必需字段")
        
        # 执行分析
        content = task["content"]
        analysis_type = task["analysis_type"]
        
        if analysis_type == "sentiment":
            result = self.analyze_sentiment(content)
        elif analysis_type == "structure":
            result = self.analyze_structure(content)
        else:
            raise ValueError(f"未知分析类型: {analysis_type}")
        
        return result
    
    def get_required_fields(self) -> list:
```

### 4.2 插件基类

#### 4.2.1 BasePlugin接口

```python
from abc import ABC, abstractmethod
from typing import Any, Dict, Optional
from pydantic import BaseModel
from enum import Enum

class PluginState(Enum):
    UNLOADED = "unloaded"
    LOADING = "loading"
    LOADED = "loaded"
    READY = "ready"
    ERROR = "error"
    DISABLED = "disabled"

class BasePlugin(ABC):
    """插件基类"""
    
    def __init__(self, plugin_id: str, config: Dict[str, Any]):
        self.plugin_id = plugin_id
        self.config = config
        self.state = PluginState.UNLOADED
        self.context = None
    
    def initialize(self, context: 'PluginContext') -> bool:
        """初始化插件"""
        try:
            self.context = context
            self.state = PluginState.LOADING
            
            # 验证配置
            if not self.validate_config():
                self.state = PluginState.ERROR
                return False
            
            # 加载资源
            if not self.load_resources():
                self.state = PluginState.ERROR
                return False
            
            self.state = PluginState.LOADED
            return True
        except Exception as e:
            self.state = PluginState.ERROR
            self.context.logger.error(f"插件初始化失败: {e}")
            return False
    
    def activate(self) -> bool:
        """激活插件"""
        if self.state != PluginState.LOADED:
            return False
        
        try:
            if not self.on_activate():
                return False
            
            self.state = PluginState.READY
            return True
        except Exception as e:
            self.state = PluginState.ERROR
            self.context.logger.error(f"插件激活失败: {e}")
            return False
    
    def deactivate(self) -> bool:
        """停用插件"""
        if self.state != PluginState.READY:
            return False
        
        try:
            if not self.on_deactivate():
                return False
            
            self.state = PluginState.LOADED
            return True
        except Exception as e:
            self.context.logger.error(f"插件停用失败: {e}")
            return False
    
    def cleanup(self) -> bool:
        """清理资源"""
        try:
            self.release_resources()
            self.state = PluginState.UNLOADED
            return True
        except Exception as e:
            self.context.logger.error(f"插件清理失败: {e}")
            return False
    
    @abstractmethod
    def execute(self, params: Dict[str, Any]) -> Dict[str, Any]:
        """执行插件功能"""
        pass
    
    def validate_config(self) -> bool:
        """验证配置"""
        return True
    
    def load_resources(self) -> bool:
        """加载资源"""
        return True
    
    def release_resources(self) -> bool:
        """释放资源"""
        return True
    
    def on_activate(self) -> bool:
        """激活时回调"""
        return True
    
    def on_deactivate(self) -> bool:
        """停用时回调"""
        return True
```

```python
# plugins/custom-analyzer-v1/plugin.py

from typing import Dict, Any
from core.plugin_interface import AnalyzerPlugin
import re

class CustomAnalyzerPlugin(AnalyzerPlugin):
    """自定义分析插件"""
    
    def __init__(self, config: Dict[str, Any]):
        super().__init__("custom-analyzer-v1", config)
    
    def analyze(self, content: str, params: Dict[str, Any]) -> Dict[str, Any]:
        """分析内容"""
        analysis_type = params.get("analysis_type", "default")
        
        if analysis_type == "word_frequency":
            return self.analyze_word_frequency(content)
        elif analysis_type == "sentence_length":
            return self.analyze_sentence_length(content)
        else:
            return self.analyze_default(content)
    
    def analyze_word_frequency(self, content: str) -> Dict[str, Any]:
        """词频分析"""
        words = re.findall(r'\w+', content.lower())
        word_count = {}
        for word in words:
            word_count[word] = word_count.get(word, 0) + 1
        
        # 排序
        sorted_words = sorted(
            word_count.items(),
            key=lambda x: x[1],
            reverse=True
        )[:10]
        
        return {
            "type": "word_frequency",
            "result": dict(sorted_words)
        }
    
    def analyze_sentence_length(self, content: str) -> Dict[str, Any]:
        """句子长度分析"""
        sentences = re.split(r'[。!?.]', content)
        lengths = [len(s.strip()) for s in sentences if s.strip()]
        
        return {
            "type": "sentence_length",
            "result": {
                "average": sum(lengths) / len(lengths) if lengths else 0,
                "max": max(lengths) if lengths else 0,
                "min": min(lengths) if lengths else 0
            }
        }
    
    def analyze_default(self, content: str) -> Dict[str, Any]:
        """默认分析"""
        return {
            "type": "default",
            "result": {
                "length": len(content),
                "word_count": len(re.findall(r'\w+', content)),
                "sentence_count": len(re.split(r'[。!?.]', content))
            }
        }
```

```python
import random
from typing import List, Dict, Any

class MultiAPIClient:
    """多API客户端"""
    
    def __init__(self, config: Dict[str, Any]):
        self.apis = config["apis"]
        self.strategy = config.get("strategy", "round_robin")
        self.current_index = 0
    
    def select_api(self) -> Dict[str, Any]:
        """选择API"""
        if self.strategy == "round_robin":
            return self.round_robin_select()
        elif self.strategy == "weighted":
            return self.weighted_select()
        elif self.strategy == "random":
            return self.random_select()
        else:
            return self.apis[0]
    
    def round_robin_select(self) -> Dict[str, Any]:
        """轮询选择"""
        api = self.apis[self.current_index]
        self.current_index = (self.current_index + 1) % len(self.apis)
        return api
    
    def weighted_select(self) -> Dict[str, Any]:
        """加权选择"""
        total_weight = sum(api["weight"] for api in self.apis)
        r = random.uniform(0, total_weight)
        
        cumsum = 0
        for api in self.apis:
            cumsum += api["weight"]
            if r <= cumsum:
                return api
        
        return self.apis[-1]
    
    def random_select(self) -> Dict[str, Any]:
        """随机选择"""
        return random.choice(self.apis)
    
    def call(self, prompt: str, **kwargs) -> str:
        """调用API"""
        api = self.select_api()
        client = self.create_client(api)
        
        try:
            response = client.chat.completions.create(
                model=api["model"],
                messages=[{"role": "user", "content": prompt}],
                **kwargs
            )
            return response.choices[0].message.content
        except Exception as e:
            # 失败时切换到下一个API
            return self.fallback_call(prompt, **kwargs)
    
    def fallback_call(self, prompt: str, **kwargs) -> str:
        """降级调用"""
        for api in self.apis:
            try:
                client = self.create_client(api)
                response = client.chat.completions.create(
                    model=api["model"],
                    messages=[{"role": "user", "content": prompt}],
                    **kwargs
                )
                return response.choices[0].message.content
            except Exception:
                continue
        
        raise RuntimeError("所有API调用失败")
```

### 6.1 缓存优化

#### 6.1.1 多级缓存

**缓存层级**:
```
L1缓存(内存) → L2缓存(磁盘) → L3缓存(数据库)
```

**缓存策略**:
- LRU(最近最少使用)
- LFU(最不经常使用)
- TTL(时间过期)

**实现代码**:
```python
from functools import lru_cache
import hashlib
import json

class MultiLevelCache:
    """多级缓存"""
    
    def __init__(self, l1_size: int = 256, l2_dir: str = "./cache"):
        self.l1_cache = {}
        self.l1_size = l1_size
        self.l2_dir = l2_dir
    
    def get(self, key: str) -> Any:
        # L1缓存
        if key in self.l1_cache:
            return self.l1_cache[key]
        
        # L2缓存
        l2_path = self._get_l2_path(key)
        if os.path.exists(l2_path):
            with open(l2_path, 'r') as f:
                value = json.load(f)
                self._set_l1(key, value)
                return value
        
        return None
    
    def set(self, key: str, value: Any, ttl: int = 3600):
        # L1缓存
        self._set_l1(key, value)
        
        # L2缓存
        l2_path = self._get_l2_path(key)
        with open(l2_path, 'w') as f:
            json.dump({"value": value, "expire": time.time() + ttl}, f)
    
    def _set_l1(self, key: str, value: Any):
        if len(self.l1_cache) >= self.l1_size:
            # LRU淘汰
            oldest_key = next(iter(self.l1_cache))
            del self.l1_cache[oldest_key]
        
        self.l1_cache[key] = value
    
    def _get_l2_path(self, key: str) -> str:
        key_hash = hashlib.md5(key.encode()).hexdigest()
        return os.path.join(self.l2_dir, f"{key_hash}.json")
```
