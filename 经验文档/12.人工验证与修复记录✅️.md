## 一、概述

### 1.1 文档目的

本文档用于指导人工验证和修复AI生成内容的质量问题，确保最终输出符合创作要求。

### 1.2 适用场景

- AI生成内容质量不达标
- 评分未通过阈值（< 0.8）
- 内容与设定冲突
- 风格一致性不足

### 2.1 自动验证结果解读

### 2.2 人工验证步骤

1. **阅读生成内容**：完整阅读AI生成的章节内容
2. **对照大纲检查**：验证主要情节是否覆盖
3. **检查人物塑造**：对话、行为是否符合人设
4. **验证世界观一致性**：设定、术语是否准确
5. **评估风格匹配**：文笔、节奏是否符合预期
6. **记录问题清单**：标注需要修复的具体位置

## 四、修复操作指南

### 4.1 修复优先级

1. **P0（阻塞性问题）**：情节遗漏、逻辑矛盾、设定冲突
2. **P1（严重问题）**：OOC、对话失真、AI痕迹
3. **P2（一般问题）**：节奏失衡、形象模糊、风格割裂
4. **P3（轻微问题）**：措辞不当、标点错误

### 4.2 修复方法

#### 4.2.1 情节修复

```
步骤：
1. 定位问题段落
2. 分析缺失/冲突的具体内容
3. 手动补充或修改情节
4. 确保与前后文衔接自然
```

**示例**：
- 问题：大纲要求"主角发现隐藏密室"，但正文直接跳过
- 修复：补充发现过程描写，包含环境描写、心理活动

#### 4.2.2 人物修复

```
步骤：
1. 对比人物设定档案
2. 标注不符合人设的对话/行为
3. 重新设计符合性格的表达
4. 保持人物弧光的一致性
```

**示例**：
- 问题：高冷角色突然话痨
- 修复：精简台词，改为简短有力的表达

#### 4.2.3 风格修复

```
步骤：
1. 对比风格学习样本
2. 识别AI痕迹特征（如"璀璨""深邃"等高频词）
3. 替换为更自然的表达
4. 调整句式节奏
```

**示例**：
- 问题："他的眼眸深邃如星空"
- 修复："他盯着她，眼睛一动不动"

#### 4.2.4 世界观修复

```
步骤：
1. 查阅世界观设定文档
2. 标注设定冲突点
3. 修正为正确设定
4. 补充必要的设定说明
```

**示例**：
- 问题：金丹期使用了元婴期的能力
- 修复：调整为金丹期合理的能力范围

### 2026-03-28 临时文件自动清理机制实现

#### 问题背景

**日志报错**:


**问题**:

2. 违反《临时文件管理规范》（记忆ID: 10093446）

#### 修复内容

**文件修改**: `core/knowledge_generator.py`

##### 新增清理方法

```python
def _cleanup_temp_files(self):
    """清理生成过程中产生的临时文件
    
    V5.4新增：遵守临时文件管理规范，生成完成后自动删除临时文件
    """
    try:
        temp_dir = self.workspace_root / "data" / "knowledge" / ".temp"
        
        if temp_dir.exists() and temp_dir.is_dir():
            # 删除临时目录下的所有文件
            for file in temp_dir.glob("*.json"):
                try:
                    file.unlink()
                    print(f"[KnowledgeGenerator] 已删除临时文件: {file.name}")
                except Exception as e:
                    print(f"[KnowledgeGenerator] 删除临时文件失败 {file.name}: {e}")
            
            # 如果目录为空，删除目录
            if not any(temp_dir.iterdir()):
                temp_dir.rmdir()
                print(f"[KnowledgeGenerator] 已删除临时目录: {temp_dir}")
                
    except Exception as e:
        print(f"[KnowledgeGenerator] 清理临时文件失败: {e}")
```

### BUG-009: main thread is not in main loop（热榜 & 生成事件）

**问题**: EventBus 使用 `ThreadPoolExecutor` 在子线程中执行所有事件 handler，handler 内直接调用 `self.root.after(0, update)` 时 Tkinter 抛出 `RuntimeError: main thread is not in main loop`。

**影响范围**: 
- `_on_hot_ranking_progress_event`（`hot_ranking.progress`）
- `_on_hot_ranking_updated_event`（`hot_ranking.updated`）
- 全部 13 个生成事件 handler（`generation.*` / `pipeline.*` / `agent.task.*`）

**根因**: `tk.after()` 内部调用 `createcommand()` 注册 Tcl 命令，该操作只能在主线程中执行。

**修复方案**: 在 `MainWindow` 辅助方法区新增 `_safe_after(ms, func)` 方法：
- 主线程：直接调用 `root.after(ms, func)`
- 子线程：将 `{"type": "_safe_after_cb", "ms": ms, "func": func}` 放入已有的 `_result_queue`，由主线程 `_process_result_queue` 轮询处理（每 100ms 一次）

**修改文件**:
- `gui_main.py`：
  - 新增 `_safe_after()` 方法（辅助方法区）
  - `_process_result_queue()` 新增 `_safe_after_cb` 分支
  - 所有 15 个事件 handler 中的 `self.root.after(0, update)` → `self._safe_after(0, update)`

**验证**: lint 检查 0 错误，方法调用点全部替换完毕。

### BUG-010: unknown option "-text"（最大化按钮）

```python
proxy_vars = ("HTTP_PROXY", "HTTPS_PROXY", "http_proxy", "https_proxy", "ALL_PROXY", "all_proxy")
saved_proxies = {k: os.environ.pop(k, None) for k in proxy_vars}
```

```python
def _setup_system_tray(self):
    """设置系统托盘图标和菜单"""
    
    # 加载图标
    icon_path = os.path.join(os.path.dirname(__file__), "icon.ico")
    icon_image = Image.open(icon_path)
    
    # 定义托盘图标菜单
    def on_show_window(icon, item):
        """显示主窗口"""
        self.root.deiconify()
        self.root.lift()
        self.root.focus_force()
        logger.info("Window restored from system tray")
    
    def on_minimize(icon, item):
        """最小化窗口"""
        try:
            self.root.withdraw()
            logger.info("Window minimized to system tray")
        except Exception as e:
            logger.error(f"Error minimizing window: {e}")
    
    def on_quit(icon, item):
        """退出程序"""
        icon.stop()
        self.root.quit()
    
    menu = Menu(
        MenuItem('显示窗口', on_show_window, default=True),
        MenuItem('最小化', on_minimize),
        MenuItem('退出', on_quit)
    )
    
    # 创建系统托盘图标（default=True让左键单击触发默认菜单项）
    icon = pystray.Icon(
        "Novel Writing Assistant",
        icon_image,
        "Novel Writing Assistant",
        menu
    )
    
    # 在后台线程运行托盘图标
    threading.Thread(target=icon.run, daemon=True).start()
```

### 技术要点

#### 1. 去重算法

**Jaccard相似度**：
```python
def calculate_similarity(self, text1: str, text2: str) -> float:
    set1 = set(text1.split())
    set2 = set(text2.split())
    intersection = set1 & set2
    union = set1 | set2
    return len(intersection) / len(union) if union else 0
```

**SimHash**：
```python
def calculate_simhash(self, text: str) -> str:
    text = text.lower().strip()
    text = ''.join(c for c in text if c.isalnum() or c in '，。、！？；：""''')
    hash_obj = hashlib.md5(text.encode('utf-8'))
    return hash_obj.hexdigest()
```
