### 4.2 修复详情

**P1-001: LanceDB API兼容性问题**

问题：`infrastructure/vector_store.py` 第761行调用 `table.size()` 方法失败
原因：LanceDB不同版本API差异
修复：改用文件系统统计物理大小

```python
# 修复前
"size_mb": table.size() / (1024 * 1024)

# 修复后
try:
    size_mb = table.size() / (1024 * 1024)
except AttributeError:
    # 使用文件系统统计
    table_path = self.db_path / f"{table_name}.lance"
    if table_path.exists():
        total_size = sum(
            os.path.getsize(f) 
            for f in table_path.rglob("*") 
            if f.is_file()
        )
        size_mb = total_size / (1024 * 1024)
    else:
        size_mb = 0
```
