### 2.2 核心方法

| 方法名 | 说明 |
|--------|------|
| `_create_consistency_content()` | 创建长篇检测面板UI |
| `_load_consistency_agent()` | 加载ConsistencyCheckerAgent |
| `_on_consistency_browse()` | 选择文件导入章节 |
| `_on_consistency_select_chapter()` | 选择已生成章节 |
| `_on_consistency_check()` | 执行一致性检查（异步） |
| `_update_consistency_result()` | 更新检测结果到UI |
| `_on_consistency_clear()` | 清空检测结果 |
| `_on_consistency_copy()` | 复制结果到剪贴板 |
| `_on_consistency_export()` | 导出检测报告 |

### 2.3 UI界面结构

```
┌────────────────────────────────────────────────────┐
│ 新章节内容                                          │
│ ┌────────────────────────────────────────────────┐ │
│ │ 文本输入框（支持文件导入/章节选择）              │ │
│ └────────────────────────────────────────────────┘ │
├────────────────────────────────────────────────────┤
│ 检测设置                                            │
│ 题材: [auto▼] 召回章节数: [10]                      │
│ 检测类型: [✓人物一致性] [✓情节连贯] [✓世界观一致]   │
├────────────────────────────────────────────────────┤
│ 检测结果                                            │
│ 状态: 就绪 | 准确率: -                              │
│ ┌────────────────────────────────────────────────┐ │
│ │ 严重程度 │ 冲突类型 │ 描述 │ 位置 │ 修复建议   │ │
│ │ 🔴严重   │ 人物     │ ...  │ ...  │ ...       │ │
│ │ 🟡中等   │ 情节     │ ...  │ ...  │ ...       │ │
│ │ 🟢轻微   │ 世界观   │ ...  │ ...  │ ...       │ │
│ └────────────────────────────────────────────────┘ │
│ 召回上下文摘要: [显示相关章节摘要]                   │
├────────────────────────────────────────────────────┤
│ [🔍 开始检测] [🗑️ 清空]        [📋 复制] [💾 导出]  │
└────────────────────────────────────────────────────┘
```

### 2.4 功能特性

1. **章节输入**
   - 支持手动粘贴文本
   - 支持选择文件导入（TXT格式）
   - 支持选择已生成章节

2. **检测设置**
   - 题材选择（auto/科幻/玄幻/历史/都市/悬疑）
   - 召回章节数（5-30章）
   - 检测类型可选（人物/情节/世界观）

3. **结果展示**
   - 冲突列表（Treeview展示）
   - 严重程度分级（P0/P1/P2）
   - 召回上下文摘要显示
   - 实时状态反馈

4. **导出功能**
   - 复制结果到剪贴板
   - 导出Markdown格式报告

## 四、集成ConsistencyCheckerAgent

### 4.1 调用方式

```python
from agents.consistency_checker_agent import ConsistencyCheckerAgent

# 创建Agent实例
agent = ConsistencyCheckerAgent()

# 构建检查上下文
context = {
    "new_chapter": "第三章 星际战争爆发...",
    "genre": "scifi",  # 可选
    "top_k": 10,  # 召回章节数
    "check_types": {
        "character": True,
        "plot": True,
        "worldview": True
    }
}

# 执行检查
result = agent.execute(context)

# 获取结果
is_consistent = result["is_consistent"]
conflicts = result["conflicts"]
suggestions = result["suggestions"]
accuracy = result["accuracy"]
```
