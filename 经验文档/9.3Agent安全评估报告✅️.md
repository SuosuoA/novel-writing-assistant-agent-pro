### 2.1 P0级问题修复（V2.0完成）

#### P0-1: 任务payload输入验证

**修复位置**: `agents/priority.py:50-151`

**问题**: 用户输入直接传递给Agent执行，无验证过滤。

**修复方案**:
- 实现Pydantic模型验证`TaskPayload`
- 添加13种危险模式检测（SQL注入、代码注入、路径遍历等）
- 支持Pydantic不可用时的简化验证

```python
class TaskPayload(BaseModel):

    @field_validator("prompt")
    @classmethod
    def validate_prompt(cls, v):
        # 检测危险模式
        dangerous_patterns = [...]
        for pattern in dangerous_patterns:
            if re.search(pattern, v, re.IGNORECASE):
                raise ValueError(f"检测到危险模式: {pattern}")
```

#### P0-2: 不可重试异常控制

**修复位置**: `agents/priority.py:38-46`, `agents/master_agent.py:292-303`

**问题**: 所有异常无限重试，导致系统崩溃。

**修复方案**:
- 定义`NON_RETRYABLE_EXCEPTIONS`常量
- MasterAgent执行前检查异常类型
- 检测到不可重试异常直接失败

```python
NON_RETRYABLE_EXCEPTIONS = (
    ValueError,
    PermissionError,
    ImportError,
    SyntaxError,
    TypeError,
    KeyError,
)

# 在execute方法中
except Exception as e:
    if isinstance(e, NON_RETRYABLE_EXCEPTIONS):
        self.status = TaskStatus.FAILED
        self.error = str(e)
        raise
```

**验证结果**: 4/4测试通过

### 2.2 P1级问题修复（V2.0完成）

#### P1-1: 全局超时控制

**修复位置**: `agents/priority.py:189-223`

**问题**: 长时间运行任务可能无限执行。

**修复方案**:
- 添加`global_timeout`属性（默认300秒）
- 实现`global_timeout_exceeded`属性检测
- MasterAgent执行前检查并跳过超时任务

```python
@property
def global_timeout_exceeded(self) -> bool:
    if self._start_time is None:
        return False
    elapsed = time.time() - self._start_time
    return elapsed > self.global_timeout
```

**验证结果**: 4/4测试通过

## 五、安全编码规范

### 5.1 输入验证规范

```python
# ✅ 正确：使用Pydantic验证
from agents.priority import TaskPayload

payload = TaskPayload(prompt=user_input, options=options)

# ❌ 错误：直接使用用户输入
execute_user_request(user_input)
```

### 5.2 异常处理规范

```python
# ✅ 正确：区分可重试和不可重试异常
from agents.priority import NON_RETRYABLE_EXCEPTIONS

try:
    result = agent.execute(task)
except NON_RETRYABLE_EXCEPTIONS as e:
    # 直接失败，不重试
    logger.error(f"不可恢复错误: {e}")
    raise
except Exception as e:
    # 可重试错误
    logger.warning(f"可重试错误: {e}")
    # 重试逻辑...

# ❌ 错误：所有异常都重试
try:
    result = agent.execute(task)
except Exception:
    retry()
```

### 5.3 敏感信息处理规范

```python
# ✅ 正确：使用安全错误处理器
from core.secure_error_handler import get_error_handler

handler = get_error_handler()
secure_error = handler.handle_error(exception, context)
log_message = handler.format_error_for_log(secure_error)

# ❌ 错误：直接记录原始异常
logger.error(f"错误: {exception}")  # 可能泄露敏感信息
```

### 5.4 权限检查规范

```python
# ✅ 正确：使用权限装饰器
from core.agent_permissions import require_permission, Permission

@require_permission(Permission.FILE_WRITE)
```
