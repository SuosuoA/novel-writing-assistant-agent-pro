## 3. V1.1新增功能

### 3.1 LLM调用超时保护

使用concurrent.futures实现强制超时保护：

```python
def _call_llm(self, system_prompt: str, user_prompt: str, 
               temperature: float = 0.8, max_tokens: int = 4000) -> str:
    with concurrent.futures.ThreadPoolExecutor(max_workers=1) as executor:
        future = executor.submit(
            self._do_api_call, system_prompt, user_prompt, 
            temperature, max_tokens
        )
        try:
            content = future.result(timeout=self._timeout)
            return content
        except concurrent.futures.TimeoutError:
            future.cancel()
            raise QuickCreationTimeoutError(f"LLM调用超时（{self._timeout}秒）")
```

### 4.5 统一入口生成

```python
from core.models import QuickCreationRequest

request = QuickCreationRequest(
    keywords="修仙、复仇、系统",
    target="all",  # worldview/outline/characters/plot/all
    genre="仙侠",
    character_count=3,
    chapter_count=20,
    detailed_level="standard"
)

result = plugin.generate_all(request)

if result.success:
    print("世界观:", result.worldview.setting_name)
    print("大纲:", result.outline.title)
    print("人物:", [c.name for c in result.characters])
    print("情节:", result.plot.plot_name)
```

**生成顺序**（确保一致性）：
1. 世界观（其他设定依赖）
2. 大纲（基于世界观）
3. 人物（基于世界观）
4. 情节（基于以上所有）

## 7. 错误处理

### 7.1 JSON解析失败

```python
def _parse_json_response(self, response: str) -> Dict[str, Any]:
    """解析JSON响应"""
    json_start = response.find("{")
    json_end = response.rfind("}") + 1
    
    if json_start >= 0 and json_end > json_start:
        json_str = response[json_start:json_end]
        try:
            return json.loads(json_str)
        except json.JSONDecodeError:
            pass
    
    return {"raw_content": response}
```
