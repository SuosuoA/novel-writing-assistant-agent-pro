## 四、P1-3：result_storage.py错误边界处理

### 4.1 问题描述

`save_result()` 和 `import_to_project()` 方法缺少文件写入异常捕获。

**影响**: 文件权限错误或磁盘满时可能崩溃

### 4.2 修复方案

添加StorageError异常类和PermissionError/OSError异常捕获。

**修复后代码**（result_storage.py）:
```python
class StorageError(Exception):
    """存储操作异常"""
    pass

def save_result(self, ...) -> Path:
    try:
        # 现有代码...
    except PermissionError as e:
        logger.error(f"文件权限错误: {e}")
        raise self.StorageError(f"无法写入文件，请检查权限: {save_path}")
    except OSError as e:
        logger.error(f"文件系统错误: {e}")
        raise self.StorageError(f"文件系统错误: {e}")
```

### 4.3 修复效果

- ✅ 文件权限错误时抛出明确异常
