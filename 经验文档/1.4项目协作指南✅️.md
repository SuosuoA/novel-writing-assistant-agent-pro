### 2.1 Python代码规范（PEP8）

#### 2.1.1 命名规范

### 2.3 Pydantic模型规范

项目使用 **Pydantic v2** 作为数据验证框架：

```python
from pydantic import BaseModel, Field, field_validator
from typing import Dict, Any
from datetime import datetime, timezone

class GenerationRequest(BaseModel):
    """生成请求模型"""
    model_config = ConfigDict(from_attributes=True)
    
    request_id: str = Field(
        default_factory=lambda: f"gen-{uuid4().hex[:8]}",
        description="请求唯一标识"
    )
    chapter_title: str = Field(..., description="章节标题")
    word_count_target: int = Field(default=2000, ge=100, le=10000)
    
    @field_validator('word_count_target')
    @classmethod
    def validate_word_count(cls, v: int) -> int:
        if v < 100 or v > 10000:
            raise ValueError('字数目标必须在100-10000之间')
        return v
```

**核心原则**：
1. 所有跨层数据使用Pydantic模型定义
2. 启用 `from_attributes=True` 支持ORM转换
3. 使用 `Field()` 提供详细描述和验证规则
4. 使用 `timezone-aware datetime` 避免时区歧义

### 2.4 线程安全规范

**Tkinter线程安全**：
```python
# ❌ 错误：在子线程中直接操作UI
def _on_event(event: Event):
    self.text_area.insert("1.0", event.data["content"])

# ✅ 正确：使用after()调度到主线程
def _on_event(event: Event):
    def update_ui():
        self.text_area.insert("1.0", event.data["content"])
    self.root.after(0, update_ui)
```

**共享资源保护**：
```python
import threading

class PluginRegistry:
    def __init__(self):
        self._plugins: Dict[str, BasePlugin] = {}
        self._lock = threading.RLock()  # 可重入锁
    
    def register(self, plugin: BasePlugin) -> bool:
        with self._lock:  # 所有共享资源访问必须在锁内
            self._plugins[plugin.metadata.id] = plugin
            return True
```

### 2.5 禁止事项

| 禁止项 | 原因 | 替代方案 |
|--------|------|----------|
| 全局变量 | 线程安全隐患 | 使用ServiceLocator依赖注入 |
| 硬编码路径 | 可移植性差 | 使用配置文件或环境变量 |
| `import *` | 命名空间污染 | 显式导入 |
| 裸`except` | 隐藏错误 | 指定具体异常类型 |
| 直接修改核心模块 | 架构稳定性 | 通过插件扩展 |

```bash
# 1. 创建发布分支

# 3. 合并到master
git checkout master
git merge release/v6.0.0
git tag -a v6.0.0 -m "Release version 6.0.0"

# 4. 合并回develop
git checkout develop
git merge release/v6.0.0

# 5. 删除发布分支
git branch -d release/v6.0.0
```

from transformers import AutoTokenizer, AutoModelForCausalLM
import torch

tokenizer = AutoTokenizer.from_pretrained("F:\\Qwen")
model = AutoModelForCausalLM.from_pretrained(
    "F:\\Qwen",
    device_map="auto",
    max_memory={0: "5GiB", "cpu": "16GiB"},
