## 1. 生成过程监控功能

### 1.1 功能说明

生成过程监控功能集成在「开始创作」页面中，实时监控小说生成过程中的Agent调度状态。

### 1.2 核心特性

- **事件订阅**：监听11种EventBus事件
  - `generation.started` - 生成开始
  - `generation.completed` - 生成完成
  - `generation.failed` - 生成失败
  - `generation.progress` - 生成进度更新
  - `pipeline.stage_started` - 阶段开始
  - `pipeline.stage_completed` - 阶段完成
  - `pipeline.iteration_started` - 迭代开始
  - `pipeline.completed` - 流水线完成
  - `agent.task.started` - Agent任务开始
  - `agent.task.completed` - Agent任务完成
  - `agent.task.failed` - Agent任务失败

- **UI组件**（位于开始创作页面的"生成过程"区域）：
  - 状态栏显示当前状态
  - 进度条显示总体进度
  - Agent状态树形视图（TreeView）
  - 日志输出区域（带滚动）

### 1.3 代码实现

事件订阅集成到 MainWindow 类中：

```python
class MainWindow:
    def __init__(self):
        # ...
        self._subscription_ids: List[str] = []
        # ...
    
    def _init_core_services(self):
        # ...
        # 订阅生成事件
        self._subscribe_generation_events()
    
    def _subscribe_generation_events(self) -> None:
        """订阅生成相关事件，绑定到开始创作页面"""
        event_bus = get_event_bus()
        if not event_bus:
            return
        
        events = [
            ("generation.started", self._on_gen_event_started),
            ("generation.completed", self._on_gen_event_completed),
            ("generation.failed", self._on_gen_event_failed),
            ("generation.progress", self._on_gen_event_progress),
            ("pipeline.stage_started", self._on_gen_event_stage_started),
            ("pipeline.stage_completed", self._on_gen_event_stage_completed),
            ("pipeline.iteration_started", self._on_gen_event_iteration_started),
            ("pipeline.completed", self._on_gen_event_pipeline_completed),
            ("agent.task.started", self._on_gen_event_agent_started),
            ("agent.task.completed", self._on_gen_event_agent_completed),
            ("agent.task.failed", self._on_gen_event_agent_failed),
        ]
        
        for event_type, handler in events:
            sub_id = event_bus.subscribe(event_type, handler)
            self._subscription_ids.append(sub_id)
    
    def _on_gen_event_agent_started(self, event) -> None:
        """Agent任务开始事件"""
        def update():
            agent_name = event.data.get("agent", "Unknown")
            self._gen_log_insert(f"[{self._timestamp()}] 🤖 Agent启动: {agent_name}\n")
            # 更新Agent状态树
            if hasattr(self, '_gen_agent_tree') and self._gen_agent_tree:
                self._gen_agent_tree.insert("", tk.END, values=("🔄 运行中", agent_name))
        
        self.root.after(0, update)
    
    def _on_gen_event_agent_completed(self, event) -> None:
        """Agent任务完成事件"""
        def update():
            agent_name = event.data.get("agent", "Unknown")
            success = event.data.get("success", False)
            status = "✅" if success else "❌"
            self._gen_log_insert(f"[{self._timestamp()}] {status} Agent完成: {agent_name}\n")
            # 更新Agent状态树
            if hasattr(self, '_gen_agent_tree') and self._gen_agent_tree:
                item_id = task_id or agent_name
                try:
                    self._gen_agent_tree.item(item_id, values=(f"{status} 完成", agent_name))
                except Exception:
                    pass
        
        self.root.after(0, update)
```

## 2. UI按钮与Agent模块连接

### 2.1 连接关系表

| UI按钮 | Agent模块 | 调用方法 |
|--------|----------|---------|
| 大纲解析 | OutlineAnalysisAgent | `agent.execute()` |
| 风格解析 | StyleLearningAgent | `agent.analyze_style()` |
| 世界观导入 | WorldviewParserAdapter | `adapter.parse_worldview()` |
| 人设批量导入 | CharacterManagerAdapter | `adapter.batch_import()` |

### 2.2 实现示例

```python
def _on_style_analyze(self):
    """风格解析按钮回调"""
    async def run_analysis():
        try:
            agent = StyleLearningAgent()
            result = await agent.analyze_style(self._style_sample_text)
            self.root.after(0, lambda: self._on_style_result(result))
        except Exception as e:
            self.root.after(0, lambda: self._on_error(str(e)))
    
    asyncio.create_task(run_analysis())

def _on_worldview_import(self):
    """世界观导入按钮回调"""
    async def run_import():
        try:
            adapter = WorldviewParserAdapter()
            result = await adapter.parse_worldview(self._worldview_text)
            self.root.after(0, lambda: self._on_worldview_result(result))
        except Exception as e:
            self.root.after(0, lambda: self._on_error(str(e)))
    
    asyncio.create_task(run_import())

def _on_character_batch_import(self):
    """人设批量导入按钮回调"""
    async def run_import():
        try:
            adapter = CharacterManagerAdapter()
            result = await adapter.batch_import(self._character_texts)
            self.root.after(0, lambda: self._on_character_result(result))
        except Exception as e:
            self.root.after(0, lambda: self._on_error(str(e)))
    
    asyncio.create_task(run_import())

def _on_outline_parse(self):
    """大纲解析按钮回调"""
    async def run_parse():
        try:
            agent = OutlineAnalysisAgent()
            result = await agent.execute({
                "outline_text": self._outline_text
            })
            self.root.after(0, lambda: self._on_outline_result(result))
        except Exception as e:
            self.root.after(0, lambda: self._on_error(str(e)))
    
    asyncio.create_task(run_parse())
```

## 3. 生成中断功能

### 3.1 实现原理

通过 `NovelGenerationService.cancel_generation()` 方法设置取消标志，各Agent在执行过程中检查该标志并优雅退出。

### 3.2 代码实现

```python
# agents/novel_generation_service.py

class NovelGenerationService:
    _cancel_flag = False
    
    @classmethod
    def cancel_generation(cls):
        """设置取消标志"""
        cls._cancel_flag = True
    
    @classmethod
    def is_cancelled(cls) -> bool:
        """检查是否已取消"""
        return cls._cancel_flag
    
    @classmethod
    def reset_cancel_flag(cls):
        """重置取消标志"""
        cls._cancel_flag = False
```

### 3.3 Agent中的检查

```python
# 在Agent执行循环中检查取消标志

async def generate_chapter(self, chapter_num):
    for paragraph in range(total_paragraphs):
        # 检查取消标志
        if NovelGenerationService.is_cancelled():
            self.logger.info("生成被用户中断")
            return None
        
        # 继续生成
        result = await self._generate_paragraph(paragraph)
```

## 4. 线程安全机制

### 4.1 核心规则

**所有UI操作必须通过 `root.after(0, callback)` 调度到主线程执行**

### 4.2 EventBus回调处理

```python
def _subscribe_generation_events(self):
    """订阅事件时确保线程安全"""
    for event_type, handler in events:
        event_bus.subscribe(event_type, handler)  # handler内部使用root.after(0, update)
```

### 4.3 异步任务回调

```python
async def run_async_task():
    result = await some_agent.execute()
    
    # 错误方式：直接修改UI
    # self.label.config(text=result)
    
    # 正确方式：调度到主线程
    self.root.after(0, lambda: self.label.config(text=result))
```
