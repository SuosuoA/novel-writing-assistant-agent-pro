## 一、架构总览

### 1.1 架构愿景

Novel Writing Assistant-Agent Pro 采用**微内核架构**，核心系统仅提供基础调度服务（EventBus、PluginRegistry、ServiceLocator、ConfigManager），所有业务功能以插件形式存在。这种架构实现了：

- **高内聚低耦合**：模块间依赖最小化，循环依赖=0
- **热插拔支持**：插件动态加载/卸载，开发环境支持热重载
- **可扩展性**：新增功能无需修改核心代码
- **可维护性**：清晰的模块边界，便于测试和调试

**分层架构说明**：
- **UI层**：用户交互界面，基于Tkinter实现，所有UI操作必须在主线程执行
- **应用层**：业务逻辑协调，由MasterAgent和专家Agent组成
- **微内核核心层**：提供EventBus、PluginRegistry、ServiceLocator、ConfigManager等基础调度服务
- **插件系统层**：所有业务功能以插件形式实现（Analyzer、Generator、Validator、Tool等）
- **服务层**：提供AI API、文件操作、记忆管理等共享服务
- **基础设施层**：提供日志、监控、数据库、熔断器等基础能力

### 1.3 核心设计原则

#### 1.3.1 微内核原则

**定义**：核心系统仅提供最基本的服务（插件调度、事件分发、依赖管理），所有业务功能通过插件实现。

**核心服务清单**：

| 服务 | 说明 | 接口 |
|------|------|------|
| EventBus | 插件间通信 | `subscribe()`, `publish()`, `publish_sync()` |
| PluginRegistry | 插件管理 | `register()`, `unregister()`, `get_plugin()`, `activate()`, `deactivate()`, `recover()` |
| ServiceLocator | 依赖注入 | `register_service()`, `get_service()` |
| ConfigManager | 配置管理 | `get_config()`, `set_config()` |

**非核心服务（插件化）**：
- 大纲解析、风格学习、人物管理、世界观解析
- 上下文构建、迭代生成、加权验证
- 热榜功能、统计分析

#### 1.3.2 榫卯式集成原则

**核心理念**：插件与核心通过标准接口"咬合"，接口定义严格，实现灵活。

**接口设计原则**：
1. **最小化接口**：只定义必要的方法，避免过度设计
2. **向后兼容**：新增方法使用默认实现，不破坏现有插件
3. **显式依赖**：插件依赖在清单中明确声明
4. **清晰边界**：插件不直接访问核心内部状态

#### 1.3.3 事件驱动原则

**核心理念**：插件间通过事件总线通信，解耦依赖。

**事件类型**：

| 事件类型 | 说明 | 典型场景 |
|----------|------|----------|
| 系统事件 | 插件加载/卸载 | `plugin.loaded`, `plugin.unloaded`, `plugin.error`, `plugin.recovered` |
| 生成事件 | 生成流程 | `generation.started`, `generation.progress`, `generation.completed` |
| 验证事件 | 质量验证 | `validation.completed` |
| 用户事件 | 用户操作 | `user.generation_requested` |
| 配置事件 | 配置变更 | `config.changed` |

## 二、分层架构设计

### 2.1 用户界面层 (UI Layer)

**职责**：负责用户交互和数据展示

**核心组件**：
- 主窗口：Tkinter主窗口，sv_ttk现代化主题
- 工作台：Notebook多标签页界面
- 设置面板：配置管理界面
- 插件管理器：插件启用/禁用界面

**技术栈**：
- Tkinter + sv_ttk 2.2+
- Python 3.12.x

**Tkinter线程安全规范**（重要）：
- **所有UI操作必须在主线程执行**
- 异步事件的handler在独立线程池执行，禁止直接操作UI
- 需要更新UI时，使用`widget.after()`调度到主线程
- `publish_sync()`在调用线程执行，主线程调用时可直接操作UI

**通信方式**：
- UI → PluginManager：通过ServiceLocator获取插件
- UI → EventBus：发布用户事件（如`user.generation_requested`）
- EventBus → UI：订阅插件事件，通过`after()`安全更新UI状态

### 2.2 应用层 (Application Layer)

**职责**：业务逻辑协调和Agent编排

**核心组件**：
- **MasterAgent**：总控Agent，协调专家Agent协作
  - **Thinker**：深度思考推理（CoT、ToT）
  - **Optimizer**：迭代优化（反馈驱动、自我修正）
  - **Validator**：质量验证（多维度评分、规则检查）
  - **Planner**：任务规划（目标分解、依赖分析）

**Agent协作模式**：
1. **链式协作**：Planner → Thinker → Optimizer → Validator → Planner（循环）
2. **树状协作**：Planner分发任务到多个Thinker，汇总后Optimizer优化
3. **网状协作**：Planner与Validator持续验证，多个Thinker并行思考

**Agent执行约束**：

| Agent | 超时 | 重试 | 降级策略 |
|-------|------|------|----------|
| Thinker | 30s | 1次 | 跳过CoT，使用直接推理 |
| Optimizer | 20s | 1次 | 返回原始生成结果 |
| Validator | 15s | 0次 | 返回默认评分0.5 |
| Planner | 10s | 0次 | 使用上一次计划 |

**技术栈**：
- Python 3.12.x
- openai SDK 1.60+（统一LLM客户端）
- Pydantic 2.10.6（数据验证）

### 2.3 微内核核心层 (Kernel Core)

## 三、微内核核心引擎

### 3.1 事件总线 (EventBus)

#### 3.1.1 核心职责

插件间发布-订阅通信的核心机制，实现插件解耦。**必须保证线程安全**。

#### 3.1.2 关键接口

```python
# core/event_bus.py
from typing import Dict, List, Any, Callable, Optional
from dataclasses import dataclass
from abc import ABC, abstractmethod
import threading
import time
from uuid import uuid4
from concurrent.futures import ThreadPoolExecutor
from datetime import datetime, timezone

@dataclass
class Event:
    """事件对象"""
    type: str                      # 事件类型
    data: Any                      # 事件数据（支持复杂对象）
    source: Optional[str] = None    # 事件来源
    timestamp: Optional[float] = None  # 时间戳
    event_id: Optional[str] = None     # 事件ID

    def __post_init__(self):
        if self.timestamp is None:
            self.timestamp = datetime.now(timezone.utc).timestamp()
        if self.event_id is None:
            self.event_id = f"evt-{uuid4().hex[:8]}"

class EventBus:
    """事件总线 - 线程安全，真异步实现"""
    
    def __init__(self, max_workers: int = 10):
        self._handlers: Dict[str, List[Dict[str, Any]]] = {}
        self._lock = threading.RLock()
        self._subscriptions: Dict[str, str] = {}
        self._executor = ThreadPoolExecutor(max_workers=max_workers)
        self._dead_letter_queue: List[Dict] = []
    
    def subscribe(self, event_type: str, handler: Callable, 
                   handler_id: str = None) -> str:
        """订阅事件，返回订阅ID"""
        if handler_id is None:
            handler_id = f"handler-{uuid4().hex[:8]}"
        
        with self._lock:
            if event_type not in self._handlers:
                self._handlers[event_type] = []
            
            self._handlers[event_type].append({
                "id": handler_id,
                "handler": handler,
            })
            self._subscriptions[handler_id] = event_type
        
        return handler_id
    
    def unsubscribe(self, subscription_id: str) -> bool:
        """取消订阅"""
        with self._lock:
            event_type = self._subscriptions.get(subscription_id)
            if not event_type:
                return False
            
            handlers = self._handlers.get(event_type, [])
            self._handlers[event_type] = [
                h for h in handlers if h["id"] != subscription_id
            ]
            
            del self._subscriptions[subscription_id]
            return True
    
    def publish(self, event_type: str, data: Any, source: str = None):
        """发布事件 - 真异步，使用线程池执行"""
        event = Event(type=event_type, data=data, source=source)
        
        with self._lock:
            handlers = list(self._handlers.get(event_type, []))
        
        # 提交到线程池异步执行
        for handler_info in handlers:
            self._executor.submit(self._safe_invoke, handler_info, event)
    
    def _safe_invoke(self, handler_info: Dict, event: Event):
        """安全调用handler，带异常处理和超时控制"""
        handler_id = handler_info["id"]
        handler = handler_info["handler"]
        
        try:
            handler(event)
        except Exception as e:
            # 记录到日志，不传播异常
            print(f"事件处理失败 {event.type} [handler={handler_id}]: {e}")
            # 添加到死信队列
            self._dead_letter_queue.append({
                "event": event,
                "handler_id": handler_id,
                "error": str(e),
                "timestamp": time.time()
            })
    
    def publish_sync(self, event_type: str, data: Any,
                     timeout: float = 5.0) -> List[Any]:
        """同步发布事件，等待所有处理器返回结果"""
        event = Event(type=event_type, data=data)
        results = []
        
        with self._lock:
            handlers = list(self._handlers.get(event_type, []))
        
        for handler_info in handlers:
            try:
                result = handler_info["handler"](event)
                results.append(result)
            except Exception as e:
                print(f"事件处理失败 {event_type}: {e}")
        
        return results
```

**使用示例**：
```python
# 订阅事件
def on_generation_started(event: Event):
    print(f"生成开始: {event.data['chapter_title']}")

event_bus.subscribe(EVENT_GENERATION_STARTED, on_generation_started)

# 发布事件（真异步，立即返回）
event_bus.publish(EVENT_GENERATION_STARTED, {
    "chapter_title": "第一章",
    "word_count_target": 2000
}, source="UI")

# 同步发布（主线程中可安全操作UI）
results = event_bus.publish_sync(EVENT_USER_GENERATION_REQUESTED, {
    "chapter": "第一章"
})
```

#### 3.1.3 性能优化

**异步事件处理**：使用线程池处理高并发事件，publish()立即返回

**批量处理**：同类事件批量提交，减少开销

**可靠事件总线增强**：
- 失败事件最多重试3次
- 超时控制默认5秒
- 死信队列（最大1000条）
- 连续失败的handler触发熔断机制

### 3.2 插件注册表 (PluginRegistry)

#### 3.2.1 核心职责

管理所有已加载的插件，提供查询、状态转换和插槽绑定功能。

#### 3.2.2 关键接口

```python
# core/plugin_registry.py
from typing import Dict, List, Optional
from enum import Enum
import threading

class PluginState(Enum):
    """插件状态"""
    LOADED = "loaded"      # 已加载（初始化完成）
    ACTIVE = "active"      # 已激活（可执行）
    ERROR = "error"        # 错误状态
    UNLOADING = "unloading"  # 卸载中

class PluginRegistry:
    """插件注册表 - 含完整状态管理"""
    
    def __init__(self):
        self._plugins: Dict[str, BasePlugin] = {}
        self._states: Dict[str, PluginState] = {}
        self._slots: Dict[str, Optional[str]] = {}
        self._lock = threading.RLock()
        self._error_counts: Dict[str, int] = {}
        self._max_retries = 3
    
    def register(self, plugin: BasePlugin) -> bool:
        """注册插件"""
        plugin_id = plugin.metadata.id
        
        with self._lock:
            if plugin_id in self._plugins:
                return False
            
            self._plugins[plugin_id] = plugin
            self._states[plugin_id] = PluginState.LOADED
            self._error_counts[plugin_id] = 0
            return True
    
    def unregister(self, plugin_id: str) -> bool:
        """注销插件"""
        with self._lock:
            if plugin_id not in self._plugins:
                return False
            
            # 解绑插槽
            for slot_id, pid in self._slots.items():
                if pid == plugin_id:
                    self._slots[slot_id] = None
            
            del self._plugins[plugin_id]
            del self._states[plugin_id]
            del self._error_counts[plugin_id]
            return True
    
    def get_plugin(self, plugin_id: str) -> Optional[BasePlugin]:
        """获取插件实例"""
        return self._plugins.get(plugin_id)
    
    def get_plugins_by_type(self, plugin_type: PluginType) -> List[BasePlugin]:
        """按类型获取插件"""
        return [
            plugin for plugin in self._plugins.values()
            if plugin.metadata.plugin_type == plugin_type
        ]
    
    def get_state(self, plugin_id: str) -> Optional[PluginState]:
        """获取插件当前状态"""
        return self._states.get(plugin_id)
    
    def _validate_transition(self, plugin_id: str,
                            target_state: PluginState) -> bool:
        """验证状态转换是否合法"""
        valid_transitions = {
            PluginState.LOADED: {PluginState.ACTIVE, PluginState.UNLOADING, PluginState.ERROR},
            PluginState.ACTIVE: {PluginState.LOADED, PluginState.UNLOADING, PluginState.ERROR},
            PluginState.ERROR: {PluginState.LOADED, PluginState.UNLOADING},
            PluginState.UNLOADING: set(),  # 终态，下一步是移除
        }
        current = self._states.get(plugin_id)
        return target_state in valid_transitions.get(current, set())
    
    def activate(self, plugin_id: str) -> bool:
        """激活插件（LOADED → ACTIVE）"""
        with self._lock:
            if plugin_id not in self._plugins:
                return False
            if not self._validate_transition(plugin_id, PluginState.ACTIVE):
                return False
            self._states[plugin_id] = PluginState.ACTIVE
            return True
    
    def deactivate(self, plugin_id: str) -> bool:
        """停用插件（ACTIVE → LOADED）"""
        with self._lock:
            if plugin_id not in self._plugins:
                return False
            if not self._validate_transition(plugin_id, PluginState.LOADED):
                return False
            self._states[plugin_id] = PluginState.LOADED
            return True
    
    def set_error(self, plugin_id: str, error: Exception) -> bool:
        """设置插件错误状态"""
        with self._lock:
            if plugin_id not in self._plugins:
                return False
            if not self._validate_transition(plugin_id, PluginState.ERROR):
                return False
            self._states[plugin_id] = PluginState.ERROR
            self._error_counts[plugin_id] = self._error_counts.get(plugin_id, 0) + 1
            return True
    
    def recover(self, plugin_id: str) -> bool:
        """从ERROR状态恢复插件（ERROR → LOADED）"""
        with self._lock:
            if plugin_id not in self._plugins:
                return False
            if self._states.get(plugin_id) != PluginState.ERROR:
                return False
            
            # 检查重试次数
            count = self._error_counts.get(plugin_id, 0)
            if count >= self._max_retries:
                return False  # 超过最大重试次数
            
            plugin = self._plugins[plugin_id]
            try:
                success = plugin.initialize(self._context)
                if success:
                    self._states[plugin_id] = PluginState.LOADED
                    return True
            except Exception as e:
                self._error_counts[plugin_id] = count + 1
            
            return False
    
    def get_error_plugins(self) -> List[str]:
        """获取所有处于ERROR状态的插件ID"""
        with self._lock:
            return [pid for pid, state in self._states.items() if state == PluginState.ERROR]
    
    def get_slot_plugin(self, slot_id: str) -> Optional[BasePlugin]:
        """获取插槽当前绑定的插件"""
        plugin_id = self._slots.get(slot_id)
        if plugin_id:
            return self._plugins.get(plugin_id)
        return None
    
    def bind_slot(self, slot_id: str, plugin_id: str) -> bool:
        """绑定插件到插槽"""
        with self._lock:
            if plugin_id not in self._plugins:
                return False
            self._slots[slot_id] = plugin_id
            return True
```

**状态转换图（V1.1修订）**：
```
LOADED → ACTIVE        (activate成功)
ACTIVE → LOADED        (deactivate成功)
LOADED → ERROR         (initialize异常)
ACTIVE → ERROR         (execute连续失败)
ERROR → LOADED         (recover成功)
LOADED → UNLOADING     (卸载请求)
ACTIVE → UNLOADING     (卸载请求，需先deactivate)
UNLOADING → (removed)  (shutdown+cleanup完成)
```

```python
# core/service_locator.py
from typing import Dict, Type, Any, Set, Optional
import threading

class CircularDependencyError(Exception):
    """循环依赖异常"""
    pass

class ServiceLocator:
    """服务定位器 - 含循环依赖检测"""
    
    def __init__(self):
        self._services: Dict[Type, Any] = {}
        self._lock = threading.RLock()
    
    def register_service(self, service_type: Type, instance: Any):
        """注册服务实例"""
        with self._lock:
            self._services[service_type] = instance
            
            # 在锁内检测循环依赖
            if hasattr(instance, 'dependencies'):
                if self._check_circular_dependency_locked(service_type):
                    del self._services[service_type]
                    raise CircularDependencyError(
                        f"检测到循环依赖，涉及服务: {service_type.__name__}"
                    )
    
    def get_service(self, service_type: Type) -> Optional[Any]:
        """获取服务实例"""
        return self._services.get(service_type)
    
    def has_service(self, service_type: Type) -> bool:
        """检查服务是否存在"""
        return service_type in self._services
    
    def _check_circular_dependency_locked(self, service_type: Type,
                                          path: Set[Type] = None) -> bool:
        """检查循环依赖"""
        if path is None:
            path = set()
        
        if service_type in path:
            return True
        
        path.add(service_type)
        
        service = self._services.get(service_type)
        if service and hasattr(service, 'dependencies'):
            for dep_type in service.dependencies:
                if self._check_circular_dependency_locked(dep_type, path.copy()):
                    return True
        
        path.discard(service_type)
        return False
```

```python
# core/config_manager.py
from typing import Any, Dict, Optional, TYPE_CHECKING
import yaml
import json
import os
from pathlib import Path
import threading

if TYPE_CHECKING:
    from .event_bus import EventBus

# 配置验证模式
CONFIG_SCHEMA = {
    "ai.temperature": {"type": float, "range": (0.0, 2.0)},
    "ai.max_tokens": {"type": int, "range": (1, 100000)},
    "ai.timeout": {"type": int, "range": (1, 300)},
    "plugins.hot_reload_debounce": {"type": float, "range": (0.1, 10.0)},
}

class ConfigValidationError(Exception):
    """配置验证错误"""
    pass

class ConfigManager:
    """配置管理器 - 含变更通知和验证"""

    def __init__(self, config_path: str = "config.yaml", event_bus: 'EventBus' = None):
        self.config_path = Path(config_path)
        self._config: Dict[str, Any] = {}
        self._event_bus = event_bus
        self._lock = threading.RLock()
        self._write_lock = threading.Lock()  # 写锁保护
        self._load_config()
    
    def _load_config(self):
        """加载配置文件"""
        if not self.config_path.exists():
            return
        
        with open(self.config_path, 'r', encoding='utf-8') as f:
            if self.config_path.suffix in ['.yaml', '.yml']:
                self._config = yaml.safe_load(f)
            elif self.config_path.suffix == '.json':
                self._config = json.load(f)
        
        self._apply_env_overrides()
    
    def _apply_env_overrides(self):
        """应用环境变量覆盖"""
        prefix = "AGENT_"
        for key, value in os.environ.items():
            if key.startswith(prefix):
                config_key = key[len(prefix):].lower().replace('_', '.')
                self._set_nested(config_key, value)
    
    def _set_nested(self, key_path: str, value: Any):
        """设置嵌套配置"""
        keys = key_path.split('.')
        current = self._config
        for key in keys[:-1]:
            if key not in current:
                current[key] = {}
            current = current[key]
        current[keys[-1]] = value
    
    def get_config(self, key_path: str, default: Any = None) -> Any:
        """获取配置"""
        keys = key_path.split('.')
        current = self._config
        for key in keys:
            if isinstance(current, dict) and key in current:
                current = current[key]
            else:
                return default
        return current
    
    def _validate_config(self, key_path: str, value: Any):
        """验证配置值"""
        schema = CONFIG_SCHEMA.get(key_path)
        if not schema:
            return
        
        expected_type = schema.get("type")
        if expected_type and not isinstance(value, expected_type):
            raise ConfigValidationError(
                f"配置项 {key_path} 类型错误: 期望 {expected_type.__name__}, 实际 {type(value).__name__}"
            )
        
        value_range = schema.get("range")
        if value_range:
            min_val, max_val = value_range
            if not (min_val <= value <= max_val):
                raise ConfigValidationError(
                    f"配置项 {key_path} 超出范围: 期望 [{min_val}, {max_val}], 实际 {value}"
                )
    
    def set_config(self, key_path: str, value: Any):
        """设置配置"""
        # 验证配置值
        self._validate_config(key_path, value)

        with self._write_lock:
            old_value = self.get_config(key_path)
            self._set_nested(key_path, value)
            self._save_config_atomic()
        
        # 发布配置变更事件
        if self._event_bus:
            self._event_bus.publish("config.changed", {
                "key": key_path,
                "old_value": old_value,
                "new_value": value
            }, source="ConfigManager")
    
    def _save_config_atomic(self):
        """原子写入配置文件"""
        temp_path = self.config_path.with_suffix('.tmp')
        
        with open(temp_path, 'w', encoding='utf-8') as f:
            if self.config_path.suffix in ['.yaml', '.yml']:
                yaml.safe_dump(self._config, f, allow_unicode=True)
            elif self.config_path.suffix == '.json':
                json.dump(self._config, f, ensure_ascii=False, indent=2)
        
        # 原子重命名
        temp_path.replace(self.config_path)
```

#### 4.3.2 依赖解析算法（拓扑排序）

```python
# core/plugin_dependency_resolver.py
from typing import Dict, List, Set, Tuple, Optional
from enum import Enum

class DependencyError(Enum):
    CIRCULAR = "circular_dependency"
    MISSING = "missing_dependency"
    VERSION_CONFLICT = "version_conflict"
    CONFLICT_PLUGIN = "conflicting_plugin"

class DependencyResolver:
    """插件依赖解析器 - 拓扑排序"""
    
    def resolve(self, plugins: Dict[str, PluginMetadata]) -> Tuple[List[str], List[Dict]]:
        """
        解析插件依赖，返回加载顺序和错误列表
        使用Kahn算法进行拓扑排序
        """
        # 构建依赖图
        graph: Dict[str, Set[str]] = {}  # plugin_id -> dependencies
        in_degree: Dict[str, int] = {}
        errors: List[Dict] = []
        
        # 初始化入度为依赖数量
        for plugin_id, metadata in plugins.items():
            graph[plugin_id] = set(metadata.dependencies)
            in_degree[plugin_id] = len(metadata.dependencies)
        
        # 检查缺失依赖
        for plugin_id, deps in graph.items():
            for dep_id in deps:
                if dep_id not in graph:
                    errors.append({
                        "type": DependencyError.MISSING,
                        "plugin": plugin_id,
                        "dependency": dep_id
                    })
        
        # 拓扑排序：入度为0的节点（无未满足依赖）先加载
        queue = [pid for pid, degree in in_degree.items() if degree == 0]
        sorted_plugins = []
        
        while queue:
            current = queue.pop(0)
            sorted_plugins.append(current)
            
            # 遍历依赖当前插件的其他插件
            for plugin_id, deps in graph.items():
                if current in deps:
                    in_degree[plugin_id] -= 1
                    if in_degree[plugin_id] == 0:
                        queue.append(plugin_id)
        
        # 检测循环依赖
        if len(sorted_plugins) != len(plugins):
            remaining = set(plugins.keys()) - set(sorted_plugins)
            errors.append({
                "type": DependencyError.CIRCULAR,
                "plugins": list(remaining)
            })
        
        return sorted_plugins, errors
```

#### 4.3.3 热插拔机制

**实现方式**：使用watchdog监控插件目录变化

**关键特性**：

**热插拔权限控制分级策略（V1.2新增）**：

| 安全等级 | 权限范围 | 可操作范围 | 适用场景 |
|----------|----------|------------|----------|
| L0-信任 | 完全信任 | 所有操作 | 官方插件、系统核心 |
| L1-受限 | 受限操作 | 沙箱内执行 | 审核通过第三方插件 |
| L2-隔离 | 最小权限 | 进程隔离执行 | 未知来源插件 |
| L3-禁止 | 无权限 | 禁止加载 | 风险插件 |

```python
class HotSwapPermission:
    """热插拔权限控制"""
    
    SECURITY_LEVELS = {
        "L0": {"can_reload": True, "can_load": True, "can_unload": True, "process_isolation": False},
        "L1": {"can_reload": True, "can_load": True, "can_unload": False, "process_isolation": False},
        "L2": {"can_reload": False, "can_load": True, "can_unload": False, "process_isolation": True},
        "L3": {"can_reload": False, "can_load": False, "can_unload": False, "process_isolation": False},
    }
    
    def __init__(self, plugin_id: str, signature_verified: bool = False):
        self.plugin_id = plugin_id
        self._security_level = self._determine_level(signature_verified)
    
    def _determine_level(self, signature_verified: bool) -> str:
        """确定插件安全等级"""
        if not signature_verified:
            return "L3"  # 未签名禁止加载
        # 已签名插件根据来源确定等级
        # 此处可根据实际需求扩展
        return "L1"  # 默认信任级别
    
    def can_reload(self) -> bool:
        return self.SECURITY_LEVELS[self._security_level]["can_reload"]
    
    def can_load(self) -> bool:
        return self.SECURITY_LEVELS[self._security_level]["can_load"]
    
    def requires_isolation(self) -> bool:
        return self.SECURITY_LEVELS[self._security_level]["process_isolation"]
```

**加载策略**：
1. 优先加载高安全级别插件（L0 > L1 > L2）
2. L2级别插件使用独立进程加载
3. L3级别插件直接拒绝加载并记录日志
4. 热重载仅允许L0/L1级别插件

```python
# infrastructure/database_migration.py
import sqlite3
import shutil
from pathlib import Path
from typing import Dict, Callable
from datetime import datetime

class DatabaseMigration:
    """数据库迁移管理器"""
    
    SCHEMA_VERSION_KEY = "schema_version"
    CURRENT_VERSION = 1
    
    def __init__(self, db_path: str):
        self.db_path = db_path
        self._backup_dir = Path(db_path).parent / "db_backups"
        self._backup_dir.mkdir(exist_ok=True)
        self._migrations: Dict[int, Callable] = {
            1: self._migration_v1,
        }
    
    def get_current_version(self) -> int:
        """获取当前数据库版本"""
        try:
            with sqlite3.connect(self.db_path) as conn:
                cursor = conn.execute(
                    "SELECT version FROM schema_version ORDER BY version DESC LIMIT 1"
                )
                row = cursor.fetchone()
                return row[0] if row else 0
        except sqlite3.OperationalError:
            return 0
    
    def backup_database(self) -> str:
        """迁移前备份"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        backup_path = self._backup_dir / f"db_backup_{timestamp}.db"
        shutil.copy2(self.db_path, backup_path)
        return str(backup_path)
    
    def migrate(self) -> bool:
        """执行迁移（带备份和回滚）"""
        try:
            current_version = self.get_current_version()
            if current_version >= self.CURRENT_VERSION:
                return True
            
            # 迁移前备份
            backup_path = self.backup_database()
            
            for version in range(current_version + 1, self.CURRENT_VERSION + 1):
                migration = self._migrations.get(version)
                if not migration:
                    raise ValueError(f"Migration version {version} not defined")
                
                print(f"Executing migration v{version}...")
                migration()
                
                # 更新版本记录
                with sqlite3.connect(self.db_path) as conn:
                    conn.execute(
                        "INSERT INTO schema_version (version) VALUES (?)",
                        (version,)
                    )
                    conn.commit()
            
            print(f"Migration completed: v{current_version} → v{self.CURRENT_VERSION}")
            return True
        
        except Exception as e:
            print(f"Migration failed: {e}")
            # 回滚：恢复备份
            if 'backup_path' in locals():
                shutil.copy2(backup_path, self.db_path)
                print(f"Rollback completed using backup: {backup_path}")
            return False
    
    def _migration_v1(self):
        """V1初始化"""
        with sqlite3.connect(self.db_path) as conn:
            # schema_version表
            conn.execute("""
                CREATE TABLE IF NOT EXISTS schema_version (
                    version INTEGER PRIMARY KEY,
                    applied_at TEXT DEFAULT (datetime('now'))
                )
            """)
            
            # generation_history表
            conn.execute("""
                CREATE TABLE IF NOT EXISTS generation_history (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    chapter_title TEXT NOT NULL,
                    iteration_count INTEGER DEFAULT 1,
                    final_score REAL,
                    word_count INTEGER,
                    created_at TEXT DEFAULT (datetime('now')),
                    metadata TEXT
                )
            """)
            
            # plugin_state表
            conn.execute("""
                CREATE TABLE IF NOT EXISTS plugin_state (
                    plugin_id TEXT PRIMARY KEY,
                    state TEXT NOT NULL,
                    enabled INTEGER DEFAULT 1,
                    load_count INTEGER DEFAULT 0,
                    last_loaded TEXT,
                    config TEXT,
                    error_count INTEGER DEFAULT 0,
                    last_error TEXT
                )

            # operation_log表
            conn.execute("""
                CREATE TABLE IF NOT EXISTS operation_log (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    operation TEXT NOT NULL,
                    module TEXT,
                    status TEXT,
                    duration_ms INTEGER,
                    detail TEXT,
                    created_at TEXT DEFAULT (datetime('now'))

            # V1.1：添加索引
```

**SQLite连接池实现**：

```python
# infrastructure/database_pool.py
import sqlite3
import threading
from queue import Queue, Empty
from typing import Optional
from contextlib import contextmanager

class SQLiteConnectionPool:
    """SQLite连接池"""
    
    def __init__(self, db_path: str, max_connections: int = 10):
        self.db_path = db_path
        self.max_connections = max_connections
        self._pool: Queue[sqlite3.Connection] = Queue(maxsize=max_connections)
        self._lock = threading.Lock()
        self._active_connections = 0
        
        # 预创建连接
        for _ in range(max_connections):
            conn = self._create_connection()
            self._pool.put(conn)
    
    def _create_connection(self) -> sqlite3.Connection:
        """创建数据库连接并设置优化参数"""
        conn = sqlite3.connect(self.db_path, check_same_thread=False)
        
        # WAL模式优化
        conn.execute('PRAGMA journal_mode=WAL')
        conn.execute('PRAGMA synchronous=NORMAL')
        conn.execute('PRAGMA foreign_keys=ON')
        
        # 性能优化PRAGMA
        conn.execute('PRAGMA cache_size=-64000')  # 64MB缓存
        conn.execute('PRAGMA temp_store=MEMORY')
        conn.execute('PRAGMA mmap_size=268435456')  # 256MB内存映射
        conn.execute('PRAGMA page_size=4096')
        
        return conn
    
    def get_connection(self, timeout: float = 30.0) -> sqlite3.Connection:
        """获取连接"""
        try:
            conn = self._pool.get(timeout=timeout)
            with self._lock:
                self._active_connections += 1
            return conn
        except Empty:
            raise TimeoutError(f"Connection pool exhausted. Active: {self._active_connections}")
    
    def release_connection(self, conn: sqlite3.Connection):
        """释放连接"""
        try:
            self._pool.put_nowait(conn)
        except:
            conn.close()
        finally:
            with self._lock:
                self._active_connections -= 1
    
    @contextmanager
    def connection(self, timeout: float = 30.0):
        """连接上下文管理器"""
        conn = self.get_connection(timeout)
        try:
            yield conn
        finally:
            self.release_connection(conn)
```

```python
# core/log_sanitizer.py
import re
from typing import Dict, Any

class LogSanitizer:
    """日志脱敏器"""
    
    SENSITIVE_FIELDS = [
        "api_key", "apikey", "api-key", "secret", "password",
        "token", "auth", "authorization", "key"
    ]
    
    API_KEY_PATTERN = re.compile(r'(sk-[a-zA-Z0-9]{20,})')
    
    def sanitize(self, data: Dict[str, Any]) -> Dict[str, Any]:
        """递归脱敏字典数据"""
        if isinstance(data, dict):
            return {k: self._sanitize_value(k, v) for k, v in data.items()}
        return data
    
    def _sanitize_value(self, key: str, value: Any) -> Any:
        """根据键名脱敏值"""
        if isinstance(value, str):
            # 检查键名是否敏感
            if any(sf in key.lower() for sf in self.SENSITIVE_FIELDS):
                return self._mask_string(value)
            # 检查值是否包含API密钥格式
            if self.API_KEY_PATTERN.search(value):
                return self.API_KEY_PATTERN.sub(r'[MASKED]', value)
        elif isinstance(value, dict):
            return self.sanitize(value)
        elif isinstance(value, list):
            return [self.sanitize(item) if isinstance(item, dict) else item for item in value]
        return value
    
    def _mask_string(self, s: str) -> str:
        """掩码字符串，保留前4位和后4位"""
        if len(s) <= 8:
            return "*" * len(s)
        return s[:4] + "*" * (len(s) - 8) + s[-4:]
```

### 10.3 熔断器设计

```python
# core/circuit_breaker.py
import time
from enum import Enum
from typing import Callable, Any

class CircuitState(Enum):
    CLOSED = "closed"      # 正常状态
    OPEN = "open"          # 熔断状态
    HALF_OPEN = "half_open"  # 半开状态

class CircuitBreaker:
    """熔断器"""
    
    def __init__(self, failure_threshold: int = 5, 
                 recovery_timeout: int = 60,
                 half_open_max_calls: int = 3):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.half_open_max_calls = half_open_max_calls
        
        self.state = CircuitState.CLOSED
        self.failure_count = 0
        self.last_failure_time = None
        self.half_open_calls = 0
    
    def call(self, func: Callable, *args, **kwargs) -> Any:
        """执行带熔断保护的函数"""
        if self.state == CircuitState.OPEN:
            if self._should_attempt_reset():
                self.state = CircuitState.HALF_OPEN
                self.half_open_calls = 0
            else:
                raise CircuitBreakerOpenException("Circuit breaker is OPEN")
        
        if self.state == CircuitState.HALF_OPEN:
            if self.half_open_calls >= self.half_open_max_calls:
                raise CircuitBreakerOpenException("Half-open call limit reached")
            self.half_open_calls += 1
        
        try:
            result = func(*args, **kwargs)
            self._on_success()
            return result
        except Exception as e:
            self._on_failure()
            raise
    
    def _should_attempt_reset(self) -> bool:
        """判断是否应尝试恢复"""
        if not self.last_failure_time:
            return False
        elapsed = time.time() - self.last_failure_time
        return elapsed >= self.recovery_timeout
    
    def _on_success(self):
        """成功回调"""
        if self.state == CircuitState.HALF_OPEN:
            self.state = CircuitState.CLOSED
            self.failure_count = 0
            self.half_open_calls = 0
        else:
            self.failure_count = max(0, self.failure_count - 1)
    
    def _on_failure(self):
        """失败回调"""
        self.failure_count += 1
        self.last_failure_time = time.time()
        
        if self.state == CircuitState.HALF_OPEN:
            self.state = CircuitState.OPEN
        elif self.failure_count >= self.failure_threshold:
            self.state = CircuitState.OPEN

class CircuitBreakerOpenException(Exception):
    """熔断器打开异常"""
    pass
```

```python
import hashlib
import hmac
from cryptography.hazmat.primitives import hashes, serialization
from cryptography.hazmat.primitives.asymmetric import rsa
from cryptography.hazmat.backends import default_backend
import json
from pathlib import Path

class PluginSigner:
    """插件代码签名器"""
    
    def __init__(self, private_key_path: str = None):
        self._private_key = None
        self._public_key = None
        if private_key_path:
            self._load_key(private_key_path)
    
    def _load_key(self, key_path: str):
        """加载密钥对"""
        with open(key_path, 'rb') as f:
            self._private_key = serialization.load_pem_private_key(
                f.read(), password=None, backend=default_backend()
            )
        self._public_key = self._private_key.public_key()
    
    def sign_plugin(self, plugin_dir: Path) -> str:
        """对插件目录进行签名"""
        # 1. 计算插件文件哈希
        manifest = self._calculate_file_hashes(plugin_dir)
        
        # 2. 将manifest序列化为字符串
        manifest_str = json.dumps(manifest, sort_keys=True)
        
        # 3. 使用私钥签名
        signature = self._private_key.sign(
            manifest_str.encode(),
            hashes.SHA256()
        )
        
        # 4. 保存签名文件
        signature_path = plugin_dir / "plugin.sig"
        signature_path.write_bytes(signature)
        
        return signature.hex()
    
    def _calculate_file_hashes(self, plugin_dir: Path) -> dict:
        """计算插件目录中所有文件的SHA256哈希"""
        manifest = {}
        for file_path in plugin_dir.rglob("*"):
            if file_path.is_file() and file_path.name not in ["plugin.sig", "plugin.pub"]:
                relative_path = file_path.relative_to(plugin_dir)
                with open(file_path, "rb") as f:
                    content = f.read()
                    manifest[str(relative_path)] = hashlib.sha256(content).hexdigest()
        return manifest
    
    def verify_signature(self, plugin_dir: Path, public_key_pem: bytes) -> bool:
        """验证插件签名"""
        signature_path = plugin_dir / "plugin.sig"
        if not signature_path.exists():
            return False
        
        # 加载公钥
        public_key = serialization.load_pem_public_key(
            public_key_pem, backend=default_backend()
        )
        
        # 重新计算哈希
        manifest = self._calculate_file_hashes(plugin_dir)
        manifest_str = json.dumps(manifest, sort_keys=True)
        
        # 验证签名
        signature = signature_path.read_bytes()
        try:
            public_key.verify(signature, manifest_str.encode(), hashes.SHA256())
            return True
        except Exception:
            return False
```

### 12.2 签名验证流程

```python
# core/plugin_loader.py - 加载时验证签名
class PluginLoader:
    """插件加载器 - 含签名验证"""
    
    TRUSTED_SIGNERS = [
        # 项目官方签名公钥
        "-----BEGIN PUBLIC KEY-----\n...\n-----END PUBLIC KEY-----"
    ]
    
    def load_plugin(self, plugin_id: str) -> BasePlugin:
        plugin_dir = self._plugin_dirs[plugin_id]
        
        # 1. 检查签名文件是否存在
        sig_path = plugin_dir / "plugin.sig"
        if not sig_path.exists():
            if self._require_signature:
                raise PluginSignatureError(f"插件 {plugin_id} 缺少签名")
            # 允许加载未签名插件（开发模式）
            return self._load_unsigned(plugin_dir)
        
        # 2. 验证签名
        signer = PluginSigner()
        for trusted_pubkey in self.TRUSTED_SIGNERS:
            if signer.verify_signature(plugin_dir, trusted_pubkey.encode()):
                break
        else:
            raise PluginSignatureError(f"插件 {plugin_id} 签名验证失败")
        
        # 3. 加载插件
        return self._load_plugin_impl(plugin_dir)
```

### ADR-006：采用ThreadPoolExecutor实现EventBus异步

**状态**：已接受

**背景**：EventBus需要异步发布事件以避免阻塞主线程，Python异步方案有ThreadPoolExecutor和asyncio两种。

**决策**：使用ThreadPoolExecutor。

**原因**：
1. 避免引入asyncio依赖（保持零额外依赖原则）
2. 与Tkinter线程模型天然兼容
3. 代码逻辑清晰，易于理解和调试
4. 无需修改现有同步代码结构

**后果**：
- 优势：实现简单、兼容性良好、调试方便
- 劣势：线程创建有开销、无法利用协程的轻量级特性

### ADR-007：SQLite + WAL模式作为数据持久化方案

**状态**：已接受

**背景**：项目需要轻量级嵌入式数据库，候选方案有SQLite、LevelDB、BoltDB。

**决策**：SQLite + WAL模式。

**原因**：
1. Python原生支持，无需额外依赖
2. WAL模式支持读写并发（Writer和Reader不互斥）
3. 成熟的生态和工具链
4. V5项目已有SQLite使用经验

**后果**：
- 优势：零依赖、并发支持好、学习成本低
- 劣势：不适合超大规模数据、需要定期维护WAL文件

### ADR-008：采用动态导入作为插件加载机制

**状态**：已接受

**背景**：需要灵活的插件加载机制，候选方案有动态导入、进程隔离、容器化。

**决策**：采用动态导入作为主要插件加载机制。

**原因**：
1. Python原生支持，实现简单
2. 性能开销最小
3. 与项目零额外依赖原则一致
4. 特殊场景（如第三方插件）可使用进程隔离

**后果**：
- 优势：实现简单、性能高、易于调试
- 劣势：插件错误可能影响主进程、隔离性不如进程隔离

### 15.1 高风险项

#### 15.1.1 ThreadPoolExecutor线程泄漏

**描述**：EventBus使用的ThreadPoolExecutor在异常情况下可能未正确关闭，导致线程资源泄漏。

**影响**：长时间运行后线程数持续增长，最终耗尽系统资源。

**缓解措施**：
```python
# 在应用退出时清理
import atexit

class EventBus:
    def __init__(self, ...):
        self._executor = ThreadPoolExecutor(max_workers=10)
        # 注册退出钩子
        atexit.register(self.shutdown)
    
    def shutdown(self):
        """优雅关闭线程池"""
        self._executor.shutdown(wait=True, cancel_futures=True)
```

**监控指标**：
- 线程池活跃线程数
- 任务队列积压长度
- 进程总线程数

#### 15.1.2 SQLite WAL模式文件膨胀

**描述**：WAL模式下的-WAL文件和-SHM文件可能持续膨胀，占用大量磁盘空间。

**影响**：磁盘空间耗尽、应用崩溃。

**缓解措施**：
```python
# 定期执行checkpoint
import sqlite3

def truncate_wal(db_path: str):
    conn = sqlite3.connect(db_path)
    conn.execute("PRAGMA wal_checkpoint(TRUNCATE)")
    conn.close()

# 定时任务：每天凌晨3点执行
import schedule
schedule.every().day.at("03:00").do(truncate_wal, "配置/app_data.db")
```

**监控阈值**：
- WAL文件超过100MB告警
- 超过500MB自动truncate

### 15.2 中风险项

#### 15.2.1 熔断器误触发

**描述**：故障阈值设置不当可能导致熔断器频繁开启，影响可用性。

**缓解措施**：
1. 根据实际负载调优阈值（初始：失败5次开启）
2. 添加熔断器状态变更告警
3. 提供手动重置接口

#### 15.2.2 API密钥迁移

**描述**：从配置文件迁移到系统密钥环时可能失败，导致服务中断。

**缓解措施**：
1. 提供迁移脚本和回滚方案
2. 迁移前自动备份
3. 保留双写机制（兼容旧配置）

```python
    def optimize_character_profiles(self, current_profiles: Dict, written_chapters: List[str]) -> Tuple[Dict, List[str]]
    def optimize_worldview_elements(self, current_worldview: Dict, written_chapters: List[str]) -> Tuple[Dict, List[str]]
```

**实现要点**：
- 调用 StyleLearner 分析已写内容的风格
- 调用 CharacterManager 检测人物行为变化

### 19.3 一键生成前期内容（PregenerationService）

**功能描述**：根据大纲自动生成创作所需的前期内容

**预设接口**：

```python
class PregenerationTask(str, Enum):
    CHARACTER_PROFILES = "character_profiles"    # 人物小传
    WORLDVIEW_ELEMENTS = "worldview_elements"    # 世界观细节
    PLOT_OUTLINE = "plot_outline"               # 情节大纲
    STYLE_GUIDE = "style_guide"                 # 风格指南
    OPENING_HOOKS = "opening_hooks"             # 开篇钩子

class PregenerationService:
    def analyze_outline(self, outline_content: str) -> Dict
    def generate_character_profiles(self, outline: str, character_names: List[str], ...) -> Dict
    def generate_worldview_elements(self, outline: str, element_types: List[str]) -> Dict
    def generate_plot_outline(self, outline: str, granularity: str = "chapter") -> PregenerationResult
    def generate_style_guide(self, outline: str, reference_texts: Optional[List[str]] = None) -> PregenerationResult
    def generate_opening_hooks(self, outline: str, style_guide: Dict, count: int = 3) -> List[PregenerationResult]
    def one_click_pregeneration(self, outline_path: str, tasks: List[PregenerationTask], ...) -> Dict
```

**实现要点**：
- 调用 OutlineParser 解析大纲
- 调用 LLM 生成人物小传、世界观细节
- 支持批量生成和一键生成
- 支持人工审核和修改
