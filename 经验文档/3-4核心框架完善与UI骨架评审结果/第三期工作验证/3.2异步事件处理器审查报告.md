### 5.2 宣集成示例验证

```python
# 异步事件处理：EventBus + AsyncHandler
from core import EventBus, AsyncHandler, init_async_handler
from core.event_bus import Event

# 初始化
event_bus = EventBus()
handler = init_async_handler(root=tk_root, worker_count=4)

# 订阅事件
def on_generate_request(event: Event):
    # 提交到异步处理器
    handler.submit(
        func=generate_content,
        args=(event.data,),
        callback=lambda r: event_bus.publish("generation.complete", r),
        error_callback=lambda e: event_bus.publish("generation.error", str(e)),
        priority=TaskPriority.HIGH,
        timeout=60.0,
    )

    event_bus.subscribe("generation.request", on_generate_request)
```

代码分析：集成示例符合架构设计，AsyncHandler可以独立使用也可以与EventBus配合使用，两者无循环依赖。
