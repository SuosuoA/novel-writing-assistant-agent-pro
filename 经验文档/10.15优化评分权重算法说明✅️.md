**功能**: 权重配置模型

**核心字段**:
```python
@dataclass
class WeightConfig:
    """权重配置数据类（V2.0版本 - 九维度智能评分）"""
    # 九维度权重（V2.0版本）
    worldview: float = 0.12          # 世界观一致性
    character: float = 0.19          # 人设一致性

    style: float = 0.19              # 风格一致性
    knowledge_reference: float = 0.08  # 知识库引用
    writing_technique: float = 0.08    # 写作技巧（V2.0新增）
    word_count: float = 0.08         # 字数符合性
    context_coherence: float = 0.08    # 上下文衔接（V2.0新增）

    enable_optimization_suggestions: bool = True  # 启用优化建议
    enable_modification_examples: bool = True     # 启用修改示例

    config_version: str = "1.1"
```

## 五、热更新机制

### 5.1 自动检测配置文件修改

**触发条件**: 每次调用`validate()`方法时

**检测逻辑**:
```python
def check_and_reload_if_modified(self) -> bool:
    """检查配置文件是否被修改，如果修改则自动重新加载"""
    if not self._config_path or not self._config_path.exists():
        return False
    
    with self._lock:
        try:
            current_mtime = self._config_path.stat().st_mtime
            
            if self._config_mtime is None or current_mtime > self._config_mtime:
                logger.info("检测到配置文件修改，触发热更新")
                return self.load_weights_from_file()
            
            return False
        except Exception as e:
            logger.error(f"检查配置文件修改失败: {e}")
            return False
```

### 5.2 API热更新

**调用方式**:
```python
# 通过插件实例热更新
plugin.update_weights(
    new_weights={
        'word_count': 0.12,
        'outline': 0.18,
        'style': 0.22,
        'character': 0.22,
        'worldview': 0.18,
        'naturalness': 0.05,
        'knowledge_consistency': 0.13,
    },
    updated_by="user_adjustment"
)
```

**热更新流程**:
```
用户调用update_weights()
    ↓
构建新的WeightConfig对象
    ↓
校验权重配置（范围、总和）
    ↓
归一化处理
    ↓
更新_weights属性
    ↓
记录更新时间和来源
    ↓
返回成功
```

### 5.3 线程安全保证

**锁机制**: `threading.RLock`

**保护范围**:
- `_weights`属性的读写
- `_config_mtime`属性的读写
- 配置文件加载和保存操作

**示例**:
```python
def get_weight(self, dimension: str) -> float:
    """获取指定维度的权重（线程安全）"""
    with self._lock:
        return self._weights.to_dict().get(dimension, 0.0)
```
