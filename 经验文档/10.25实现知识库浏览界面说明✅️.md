## 二、UI功能

### 2.1 搜索区

| 功能 | 说明 |
|------|------|
| 关键词搜索 | 支持回车触发搜索 |
| 题材过滤 | all/scifi/xuanhuan/history/general |
| 领域过滤 | all/physics/chemistry/biology/religion/mythology/philosophy/logic |
| 返回数量 | 10-200可调 |

### 2.2 统计区

- 显示总知识点数
- 各题材知识点数
- 支持查看详细统计

### 2.3 知识列表

| 列 | 说明 |
|------|------|
| ID | 知识点唯一标识 |
| 标题 | 知识点标题 |
| 题材 | 分类（scifi/xuanhuan等） |
| 领域 | 知识领域 |
| 关键词 | 前5个关键词 |
| 内容预览 | 前80字符 |

### 2.4 详情面板

- 双击知识点显示完整内容
- 包含ID、标题、题材、领域、关键词、难度、时间、内容、参考来源

### 2.5 底部按钮

| 按钮 | 功能 |
|------|------|
| 导入JSON | 批量导入知识点 |
| 导出JSON | 导出所有知识点 |
| 添加知识点 | 手动添加单个知识点 |
| 编辑选中 | 编辑选中知识点 |
| 删除选中 | 删除选中知识点 |
