## 一、概述

### 1.1 目标

将V5核心插件包装为Agent能力，实现：
- **统一接口**：所有插件通过BaseAgent统一调度
- **标准化结果**：插件输出统一转换为AgentResult
- **MasterAgent调度**：支持优先级、依赖、重试机制
- **插件解耦**：Agent层隔离插件实现细节

### 1.2 Agent化原则

| 原则 | 说明 |
|------|------|
| **零额外依赖** | 不引入新框架，复用现有Agent基础设施 |
| **适配器模式** | Agent作为插件适配器，不修改插件代码 |
| **标准接口** | 实现BaseAgent接口：initialize/execute/can_handle/cleanup |
| **结果转换** | 插件输出统一转换为AgentResult数据结构 |

### 1.3 文件结构

```
agents/
├── plugins/                      # Agent插件目录
│   ├── __init__.py              # 模块导出
│   ├── outline_analysis_agent.py    # 大纲分析Agent
│   ├── style_learning_agent.py      # 风格学习Agent
│   ├── novel_generation_agent.py    # 小说生成Agent
│   └── quality_validation_agent.py  # 质量验证Agent
├── core/
│   └── base_agent.py            # Agent基类定义
└── master_agent.py              # MasterAgent调度器
```

```

│                    MasterAgent                          │

└──────────────────────┬──────────────────────────────────┘
                       │ 调度
                       ▼
┌──────────────────────────────────────────────────────────┐
│                   AgentPool                              │
│  (Agent实例池，管理Agent生命周期)                        │
└────────┬─────────────────┬─────────────────┬────────────┘
         │                 │                 │
         ▼                 ▼                 ▼
┌─────────────┐   ┌─────────────┐   ┌─────────────┐
│ Agent 1     │   │ Agent 2     │   │ Agent N     │
│ (Adapter)   │   │ (Adapter)   │   │ (Adapter)   │
└──────┬──────┘   └──────┬──────┘   └──────┬──────┘
       │                 │                 │
       ▼                 ▼                 ▼
┌─────────────┐   ┌─────────────┐   ┌─────────────┐
│ Plugin 1    │   │ Plugin 2    │   │ Plugin N    │
```

```python
    "target_word_count": 3500,          # 可选，默认3500
    "style_profile": {...},             # 可选，风格档案
    "use_context_memory": True,         # 可选，默认True
```

**AgentResult.data结构**：
```python
{
    "content": "生成的章节内容...",      # 完整章节内容
    "word_count": 3500,                 # 实际字数
    "stats": {
        "final_score": 0.85,            # 最终评分
        "total_iterations": 3,          # 迭代次数
        "all_scores": [0.6, 0.75, 0.85],# 每次迭代评分
        "has_chapter_end": True,        # 是否包含【本章完】
    },
}
```

**核心流程（强制保护）**：
1. 整理打包请求内容（强制附加【本章完】要求）
2. 向大模型发送请求
3. 接受返回文章
4. 多维度评分
5. 分数<0.8 → 发送评分+修改建议 → 再次生成
6. 评分≥0.8且有【本章完】→ 输出保存

**依赖验证（V1.1新增）**：
```python
def _check_dependencies(self) -> list:
    """检查依赖插件是否可用

    验证dependencies字段定义的依赖插件是否已加载
    """
    missing = []
    dependencies = self._metadata.dependencies or []

    for dep_id in dependencies:
        try:
            if self._service_locator:
                dep = self._service_locator.try_get(dep_id)
                if not dep:
                    missing.append(dep_id)
            else:
                missing.append(dep_id)
        except Exception:
            missing.append(dep_id)

    return missing
```

依赖插件缺失时，Agent仍可初始化，但会记录警告日志。

## 四、MasterAgent调度集成

### 4.1 注册到AgentPool

在 `agents/agent_pool.py` 中已添加 `_register_builtin_agents()` 方法：

```python
def _register_builtin_agents(self) -> Dict[str, bool]:
    """注册内置Agent适配器

    将4个V5核心插件包装为Agent并注册到Agent池。

    Returns:
        各Agent注册结果 {agent_type: success}
    """
    from .plugins import (
        OutlineAnalysisAgent,
        StyleLearningAgent,
        NovelGenerationAgent,
        QualityValidationAgent,
    )

    results: Dict[str, bool] = {}
    adapters = [
        OutlineAnalysisAgent(),
        StyleLearningAgent(),
        NovelGenerationAgent(),
        QualityValidationAgent(),
    ]

    for agent in adapters:
        success = self.register_agent(agent)
        results[agent.agent_type] = success

    # 发布内置Agent注册完成事件
    self._event_bus.publish(
        "agent.pool.builtin_registered",
        {
            "total": len(adapters),
            "success_count": sum(1 for v in results.values() if v),
            "results": results,
        },
        source="AgentPool",
    )

    return results
```

**使用方式**：
```python
# 创建AgentPool
agent_pool = AgentPool(event_bus)

# 注册并初始化内置Agent
agent_pool._register_builtin_agents()
agent_pool.initialize()
```

### 4.2 任务类型映射

| 任务类型 | Agent类型 |
|----------|-----------|
| `outline_analysis`, `outline_parse`, `outline_extract` | `outline_analysis` |
| `style_learning`, `style_analysis`, `style_extract` | `style_learning` |
| `novel_generation`, `chapter_generation`, `generate_chapter` | `novel_generation` |
| `quality_validation`, `content_validation`, `validate_quality` | `quality_validation` |

### 4.3 链式协作示例

生成一章小说的完整流程：

```python
# 1. 大纲分析
outline_task = AgentTask(
    task_id="outline-001",
    agent_type="outline_analysis",
    priority=TaskPriority.CRITICAL,
    payload={"file_path": "大纲.md"},
)

# 2. 风格学习
style_task = AgentTask(
    task_id="style-001",
    agent_type="style_learning",
    priority=TaskPriority.CRITICAL,
    payload={"file_path": "样本.txt"},
)

# 3. 小说生成（依赖前两步）
generation_task = AgentTask(
    task_id="gen-001",
    agent_type="novel_generation",
    priority=TaskPriority.CRITICAL,
    payload={
        "chapter_title": "第一章",
        "chapter_outline": "...",
        "target_word_count": 3500,
    },
    dependencies=["outline-001", "style-001"],
)

# 4. 质量验证（依赖生成）
validation_task = AgentTask(
    task_id="val-001",
    agent_type="quality_validation",
    priority=TaskPriority.CRITICAL,
    payload={
        "content": "${gen-001.data.content}",  # 引用上一步结果
        "target_word_count": 3500,
    },
    dependencies=["gen-001"],
)

# 提交任务链
for task in [outline_task, style_task, generation_task, validation_task]:
    master_agent.submit_task(task)
```

## 五、AgentResult转换规范

### 5.1 成功结果

```python
return AgentResult.success_result(
    task_id=task_id,
    agent_type=self.AGENT_TYPE,
    data=result,                        # 插件原始输出
    metadata={                          # 元数据摘要
        "key_metrics": value,
    }
)
```

### 5.2 失败结果

```python
return AgentResult.failure_result(
    task_id=task_id,
    agent_type=self.AGENT_TYPE,
    error="错误描述",
    error_type="ErrorType",             # InitializationError/AnalysisError/ExecutionError
)
```

### 5.3 错误类型定义

| 错误类型 | 说明 |
|----------|------|
| `InitializationError` | Agent或插件初始化失败 |
| `AnalysisError` | 分析类任务失败 |
| `GenerationError` | 生成类任务失败 |
| `ValidationError` | 验证类任务失败 |
| `ExecutionError` | 执行过程中异常 |

## 六、使用示例

### 6.1 直接使用Agent

```python
from agents.plugins import OutlineAnalysisAgent

# 创建Agent
agent = OutlineAnalysisAgent()

# 初始化
agent.initialize()

# 执行任务
result = agent.execute(
    task_id="task-001",
    payload={
        "content": "第一章：开端...",
        "options": {"use_langchain": True}
    }
)

# 检查结果
if result.success:
    print(f"解析成功: {result.data['total_chapters']} 章")
else:
    print(f"解析失败: {result.error}")

# 清理
agent.cleanup()
```

### 6.2 通过MasterAgent调度

```python
from agents import MasterAgent, AgentTask, TaskPriority

# 创建MasterAgent
master = MasterAgent(event_bus)
master.start()

# 创建任务
task = AgentTask(
    task_id="task-001",

    payload={
        "content": sample_text,
        "author_name": "测试作者",

# 提交任务
task_id = master.submit_task(task)

# 等待结果（通过事件订阅）
# 或通过轮询检查任务状态
```
