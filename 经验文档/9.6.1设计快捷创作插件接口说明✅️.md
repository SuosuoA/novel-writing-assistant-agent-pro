## 一、概述

### 1.1 设计目标

快捷创作插件（QuickCreationPlugin）旨在帮助用户快速生成小说创作所需的基础设定，包括：
- 世界观设定（世界观结构、力量体系、社会背景等）
- 大纲设定（章节规划、剧情走向、高潮节点等）
- 人物设定（角色形象、性格特点、人物关系等）
- 情节设定（场景设计、冲突转折、伏笔布局等）

### 1.2 核心能力

| 能力 | 说明 |
|------|------|
| 关键词驱动 | 仅需提供关键词即可生成设定 |
| 参考文本 | 支持参考已有文本风格 |
| 单项生成 | 可单独生成某一类设定 |
| 全部生成 | 一键生成所有设定，保持一致性 |
| 结构化输出 | 返回结构化数据，便于编辑和使用 |

## 二、接口设计

### 2.1 插件继承关系

```
BasePlugin (基础插件)
    └── ToolPlugin (工具插件)
            └── QuickCreationPlugin (快捷创作插件)
```

### 2.2 核心方法

```python
class QuickCreationPlugin(ToolPlugin):
    """快捷创作插件接口"""
    
    # 核心方法（抽象方法）
    @abstractmethod
    def generate_worldview(self, request: QuickCreationRequest) -> WorldviewResult:
        """生成世界观设定"""
        pass
    
    @abstractmethod
    def generate_outline(self, request: QuickCreationRequest, worldview: Optional[WorldviewResult] = None) -> OutlineResult:
        """生成大纲设定"""
        pass
    
    @abstractmethod
    def generate_characters(self, request: QuickCreationRequest, outline: Optional[OutlineResult] = None, worldview: Optional[WorldviewResult] = None) -> List[CharacterResult]:
        """生成人物设定"""
        pass
    
    @abstractmethod
    def generate_plot(self, request: QuickCreationRequest, outline: Optional[OutlineResult] = None, characters: Optional[List[CharacterResult]] = None, worldview: Optional[WorldviewResult] = None) -> PlotResult:
        """生成情节设定"""
        pass
    
    # 统一方法（已实现）
    def generate_all(self, request: QuickCreationRequest) -> QuickCreationResult:
        """生成全部设定（按序调用，确保一致性）"""
        pass
    
    # 辅助方法
    def get_creation_options(self) -> Dict[str, Any]:
        """获取创作选项定义"""
        pass
    
    def estimate_creation(self, request: QuickCreationRequest) -> Dict[str, Any]:
        """预估创作效果"""
        pass
```

## 三、数据模型

### 3.1 请求参数（QuickCreationRequest）

```python
class QuickCreationRequest(BaseModel):
    """快捷创作请求参数"""
    
    # 必需参数
    keywords: str                    # 创作关键词
    
    # 生成目标
    target: str = "all"              # 生成目标（worldview/outline/characters/plot/all）
    
    # 参考文本（可选）
    reference_text: Optional[str]    # 参考文本
    reference_type: Optional[str]    # 参考类型（novel/outline/character/worldview）
    
    # 风格选项
    genre: Optional[str]             # 题材类型（玄幻、都市、言情等）
    style: Optional[str]             # 写作风格（轻松、严肃、热血等）
    tone: Optional[str]              # 基调（爽文、虐文、喜剧等）
    
    # 数量配置
    character_count: int = 3         # 人物数量（1-10）
    chapter_count: int = 10          # 章节数量（1-100）
    word_count_per_chapter: int = 2000  # 每章字数（500-10000）
    
    # 高级选项
    detailed_level: str = "medium"   # 详细程度（brief/medium/detailed）
    include_relationships: bool = True  # 是否包含人物关系
    include_timeline: bool = True    # 是否包含时间线
    request_id: Optional[str]        # 请求ID
```

## 四、方法调用流程

### 4.1 单项生成流程

```
用户输入关键词
    ↓
选择生成目标（worldview/outline/characters/plot）
    ↓
调用对应方法（generate_xxx）
    ↓
返回单项结果
```

### 4.2 全部生成流程

```
用户输入关键词
    ↓
调用 generate_all()
    ↓
1. generate_worldview() → 世界观
    ↓
2. generate_outline(worldview) → 大纲（基于世界观）
    ↓
3. generate_characters(outline, worldview) → 人物（基于世界观和大纲）
    ↓
4. generate_plot(outline, characters, worldview) → 情节（基于全部设定）
    ↓
返回 QuickCreationResult（包含所有设定）
```

**一致性保障**：
- 大纲生成时参考世界观，确保剧情与世界设定一致
- 人物生成时参考大纲和世界观，确保角色符合剧情需要
- 情节生成时参考全部设定，确保情节连贯完整

## 五、使用示例

### 5.1 生成世界观

```python
# 创建请求
request = QuickCreationRequest(
    keywords="玄幻、穿越、复仇、系统",
    target="worldview",
    genre="玄幻",
    detailed_level="detailed"
)

# 调用生成
result = plugin.generate_worldview(request)

# 获取文本
worldview_text = result.get_full_text()
```

### 5.2 生成全部设定

```python
# 创建请求
request = QuickCreationRequest(
    keywords="玄幻、穿越、复仇、系统",
    target="all",
    genre="玄幻",
    style="热血",
    tone="爽文",
    character_count=5,
    chapter_count=20,
    detailed_level="medium"
)

# 调用生成
result = plugin.generate_all(request)

# 获取所有文本
all_texts = result.get_all_text()
# all_texts = {
#     "worldview": "...",
#     "outline": "...",
#     "characters": "...",
#     "plot": "..."
# }
```

### 5.3 通过execute调用

```python
# 调用execute方法
result = plugin.execute("worldview", {
    "keywords": "玄幻、穿越、复仇",
    "genre": "玄幻"
})

# result = {
#     "worldview": "完整文本...",
#     "data": {...}  # 结构化数据
# }
```

## 七、实现要点

### 7.1 抽象方法必须实现

子类必须实现以下抽象方法：
- `generate_worldview(request)` → WorldviewResult
- `generate_outline(request, worldview)` → OutlineResult
- `generate_characters(request, outline, worldview)` → List[CharacterResult]
- `generate_plot(request, outline, characters, worldview)` → PlotResult

### 7.2 generate_all已提供默认实现

`generate_all()` 方法已提供默认实现，按顺序调用各生成方法，确保一致性。子类无需重写。

### 7.3 可选重写的方法

- `get_creation_options()` - 可重写以自定义选项
- `estimate_creation()` - 可重写以提供更精确的预估
- `execute()` - 已适配ToolPlugin接口，通常无需重写

## 八、与V5核心模块的关系

### 8.1 V5模块保护

快捷创作插件**不修改**任何V5核心模块，而是作为独立工具插件提供新功能。

### 8.2 可复用V5能力

实现时可复用V5核心模块的能力：
- `universal_worldview_parser.py` - 世界观解析
- `enhanced_character_manager.py` - 人物管理
- `outline_parser_v3.py` - 大纲解析
