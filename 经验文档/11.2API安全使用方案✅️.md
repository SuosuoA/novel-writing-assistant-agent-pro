### 1.1 用户需求

1. **UI设置页面**：提供本地大模型和线上API的切换接口，支持保存设置

## 三、插件实现

### 3.1 API配置管理插件 (api-config-manager)

**插件ID**: `api-config-manager-v1`

**路径**: `plugins/api-config-manager-v1/`

**核心功能**:
1. **加密存储API Key**:
   - 使用AES-256-GCM加密算法
   - 主密钥自动生成，存储在`.secrets/.master_key`
   - 加密后的API Key存储在`.secrets/api_keys.enc`

2. **全局配置管理**:
   - 维护`config.yaml`（非敏感配置）
   - 维护`.secrets/api_keys.enc`（敏感配置）
   - 两者合并后返回完整配置

3. **EventBus集成**:
   - 发布`api.config.changed`事件
   - 订阅模块实时获取新配置

**核心类**:
```python
class APIConfigManagerPlugin(ToolPlugin):
    """API配置管理插件"""
    
    PLUGIN_ID = "api-config-manager-v1"
    PLUGIN_NAME = "API配置管理器"
    PLUGIN_VERSION = "1.0.0"
    
    def __init__(self):
        self._cipher = None  # AES加密器
        self._master_key = None  # 主密钥
        self._secrets_path = Path(".secrets")
        
    def save_api_key(self, provider: str, api_key: str) -> None:
        """加密保存API Key"""
        
    def get_api_key(self, provider: str) -> Optional[str]:
        """解密获取API Key"""
        
    def get_full_config(self) -> Dict[str, Any]:
        """获取完整配置（合并config.yaml和api_keys.enc）"""
        
    def test_connection(self, provider: str, api_key: str) -> bool:
        """测试API连接"""
```

### 4.4 加密存储实现

**使用AES-256-GCM加密API Key**:

```python
from cryptography.hazmat.primitives.ciphers.aead import AESGCM
import os
import json
from pathlib import Path

class APIKeyEncryption:
    """API Key加密存储"""
    
    MASTER_KEY_PATH = Path(".secrets/.master_key")
    SECRETS_PATH = Path(".secrets/api_keys.enc")
    
    def __init__(self):
        self._ensure_secrets_dir()
        self._master_key = self._load_or_create_master_key()
        self._aesgcm = AESGCM(self._master_key)
    
    def _ensure_secrets_dir(self):
        """确保.secrets目录存在"""
        self.MASTER_KEY_PATH.parent.mkdir(parents=True, exist_ok=True)
    
    def _load_or_create_master_key(self) -> bytes:
        """加载或创建主密钥"""
        if self.MASTER_KEY_PATH.exists():
            return self.MASTER_KEY_PATH.read_bytes()
        else:
            # 生成32字节（256位）随机密钥
            key = os.urandom(32)
            self.MASTER_KEY_PATH.write_bytes(key)
            # 设置文件权限（仅用户可读写）
            os.chmod(self.MASTER_KEY_PATH, 0o600)
            return key
    
    def encrypt_and_save(self, api_keys: Dict[str, str]) -> None:
        """加密并保存API Keys"""
        # 生成随机nonce（12字节）
        nonce = os.urandom(12)
        # 加密
        plaintext = json.dumps(api_keys).encode('utf-8')
        ciphertext = self._aesgcm.encrypt(nonce, plaintext, None)
        # 保存（nonce + ciphertext）
        self.SECRETS_PATH.write_bytes(nonce + ciphertext)
        os.chmod(self.SECRETS_PATH, 0o600)
    
    def decrypt_and_load(self) -> Dict[str, str]:
        """解密并加载API Keys"""
        if not self.SECRETS_PATH.exists():
            return {}
        
        data = self.SECRETS_PATH.read_bytes()
        nonce = data[:12]
        ciphertext = data[12:]
        
        try:
            plaintext = self._aesgcm.decrypt(nonce, ciphertext, None)
            return json.loads(plaintext.decode('utf-8'))
        except Exception:
            return {}
```
