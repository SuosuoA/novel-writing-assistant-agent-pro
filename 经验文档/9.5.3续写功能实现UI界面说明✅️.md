## 一、功能概述

续写功能UI界面是Novel Writing Assistant-Agent Pro项目工作台的第八个子功能，提供智能小说续写的完整交互体验。

### 1.1 核心功能

| 功能 | 描述 |
|------|------|
| 文本输入 | 支持粘贴、文件导入、选择已生成章节 |
| 字数选择 | 预设选项（500/1000/2000）+ 自定义输入 |
| 续写方向 | 10种方向选择（自然/剧情推进/高潮铺垫等） |
| 创意温度 | 0.0-1.0滑块调节 |
| 生成模式 | 单次生成 / 多版本生成 |
| 版本管理 | 多版本存储、切换、对比、选择最佳 |
| 结果保存 | 新章节 / 追加到章节 / 导出文件 |

### 1.2 技术栈

- **GUI框架**: Tkinter + sv_ttk
- **设计风格**: Glass Morphism（玻璃态）
- **插件集成**: ContinuationGeneratorPlugin
- **异步处理**: AsyncHandler + threading
- **线程安全**: root.after(0, callback)

## 三、功能实现

### 3.1 开始续写

**流程**:

1. 参数验证（插件、原文、字数）
2. 获取上下文（大纲、人设、世界观、前文）
3. 根据模式执行：
   - 单次生成：调用 `generate_continuation()`
   - 多版本生成：调用 `generate_multiple_versions()`
4. 自动评分并选择最佳版本
5. 更新UI显示

**关键代码**:

```python
def _on_continue_generate(self):
    # 获取参数
    source_text = self._continue_source.get("1.0", tk.END).strip()
    word_count = int(self._continue_words_var.get())
    direction = direction_map.get(self._continue_direction_var.get(), "natural")
    temperature = self._continue_temp_var.get()
    is_multi_version = self._continue_mode_var.get() == "多版本生成"
    
    # 获取上下文
    outline = self._project_manager.get_outline() if self._project_manager else None
    characters = self._project_manager.get_characters() if self._project_manager else None
    # ...
    
    # 异步执行
    def generate_task():
        request = ContinuationRequest(
            starting_text=source_text,
            word_count=word_count,
            direction=direction,
            outline=outline,
            characters=characters,
            # ...
        )
        
        if is_multi_version:
            results = self._continuation_plugin.generate_multiple_versions(
                request, 
                num_versions=3
            )
            # 自动选择最佳
            best_result, best_index, scores = self._continuation_plugin.select_best_version(results)
        else:
            result = self._continuation_plugin.generate_continuation(request)
        
        # 更新UI
        self.root.after(0, lambda: self._display_version(index))
    
    # 启动后台线程
    threading.Thread(target=generate_task, daemon=True).start()
```

def _load_continuation_plugin(self):

### 5.3 版本显示

```python
def _display_version(self, index: int):
    version = self._continue_versions[index]
    
    # 更新文本框
    self._continue_result.delete("1.0", tk.END)
    self._continue_result.insert("1.0", version["text"])
    
    # 更新信息栏
    metadata = version["metadata"]
    self._result_info_label.configure(
        text=f"字数：{version['word_count']} | 耗时：{metadata['generation_time']:.1f}秒 | 模型：{metadata['model_name']}"
    )
    self._score_label.configure(text=f"评分：{version['score']:.2f}")
    
    # 更新版本信息
    is_best = index == self._best_version_index
    best_mark = " ⭐最佳" if is_best else ""
    self._version_info_label.configure(text=f"版本 {index + 1}/{len(self._continue_versions)}{best_mark}")
```
