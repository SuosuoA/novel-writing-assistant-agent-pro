## 三、专用接口评审

### 3.1 AnalyzerPlugin

```python
class AnalyzerPlugin(BasePlugin):
    @abstractmethod
    def analyze(self, content: str, options: Optional[Dict[str, Any]] = None) -> Dict[str, Any]: pass
    @abstractmethod
    def get_supported_formats(self) -> List[str]: pass
    @abstractmethod
    def get_analysis_types(self) -> List[str]: pass
```

**评审意见**: ✅ 接口简洁明确，适用于大纲解析、风格学习、人物管理、世界观解析等场景。

### 3.2 GeneratorPlugin

```python
class GeneratorPlugin(BasePlugin):
    @abstractmethod
    def generate(self, request: GenerationRequest) -> GenerationResult: pass
    @abstractmethod
    def validate_request(self, request: GenerationRequest) -> tuple[bool, List[str]]: pass
    @abstractmethod
    def get_generation_options(self) -> Dict[str, Any]: pass
    def cancel(self, request_id: str) -> bool:  # 可选实现
        return False
```

**评审意见**: ✅ 使用Pydantic模型（GenerationRequest/GenerationResult）作为入参和返回值，符合架构设计说明书V1.2的ADR-005决策。

### 3.3 ValidatorPlugin

```python
class ValidatorPlugin(BasePlugin):
    @abstractmethod
    def validate(self, content: str, context: Optional[Dict[str, Any]] = None) -> ValidationScores: pass
    @abstractmethod
    def get_validation_dimensions(self) -> List[str]: pass
```

**评审意见**: ✅ ValidationScores模型包含6维度评分和加权计算方法，与【评分反馈，循环优化生成流程】保护规则一致。
