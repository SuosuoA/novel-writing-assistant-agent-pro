### 3.1 架构设计
- **微内核架构**：EventBus + PluginRegistry + ServiceLocator三大核心，职责清晰
- **线程安全**：所有核心模块使用RLock，读拷贝快照模式避免读写冲突
- **全局单例**：双重检查锁定保证线程安全的延迟初始化

### 3.2 EventBus真异步实现
```python
# event_bus.py 第180-213行
def publish(self, event_type: str, data: Any = None, source: Optional[str] = None) -> None:
    """发布事件 - 真异步，使用线程池执行"""
    event = Event(...)
    
    # 1. 在锁内拷贝handlers快照（读拷贝模式）
    with self._lock:
        handlers = list(self._handlers.get(event_type, []))
    
    # 2. 提交到线程池异步执行（不阻塞调用线程）
    if handlers:
        self._executor.submit(self._dispatch_event, event, handlers)
    
    # 3. 立即返回，调用线程不被阻塞
```
**符合设计文档第170-195行规范，真正实现了异步发布**
