## 四、DeepSeek API集成详情

### 4.1 API调用方式

**使用OpenAI SDK**：
```python
from openai import OpenAI

# 创建DeepSeek客户端
client = OpenAI(
    api_key="YOUR_DEEPSEEK_API_KEY",  # 从config.yaml读取
    base_url="https://api.deepseek.com"
)

# 发送请求
response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[
        {"role": "system", "content": "你是一个小说写作助手。"},
        {"role": "user", "content": "帮我生成一段场景描写"}
    ],
    max_tokens=1000,
    temperature=0.7
)

# 获取回复
reply = response.choices[0].message.content
```
