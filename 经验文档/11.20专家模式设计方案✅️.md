## 一、设计目标

### 1.1 核心需求

| 需求 | 说明 | 优先级 |
|------|------|--------|
| 轻量化可插拔 | 不增加启动负担，不集成到GUI文件 | P0 |
| 专家按钮 | UI增加专家选择器，每次创作可选专家 | P0 |
| 小说创作专家 | 专门优化小说质量，整合现有数据源 | P0 |
| 智能评分反馈 | 9维度评分+优化建议，切实可执行 | P0 |
| 本地模型辅助 | 调用sentence_transformers优化评分 | P1 |
| Claw记忆集成 | 越用越聪明，自动优化 | P1 |
| 失效工作流清理 | 审核清理重叠功能 | P2 |

### 1.2 设计原则

#### 1.2.1 最小改动原则

| 原则 | 说明 |
|------|------|
| 不修改核心框架 | 保持EventBus/PluginRegistry/ServiceLocator不变 |
| 不增加启动负担 | 延迟加载，按需初始化 |
| 不破坏V5保护 | 完全继承V5核心模块保护规则 |
| 不修改GUI主文件 | 专家选择器独立模块 |

#### 1.2.2 插件化原则

```
专家模式 = ExpertPlugin (继承GeneratorPlugin)
    ├── ExpertGenerator (生成增强)
    ├── ExpertValidator (评分增强)
    ├── ExpertOptimizer (优化建议)
    └── ExpertMemory (Claw集成)
```

#### 1.2.3 向后兼容原则

- **默认行为**: 不选择专家时，使用原有生成流程
- **降级方案**: 专家插件加载失败时，回退到普通模式
- **保护机制**: 完全继承V5核心模块保护规则

```python
                "characters": enhanced_request.character_references,
                ...
            }
        )
        
        # 步骤4: 生成优化建议
        if expert_evaluation.total_score < 0.8:
            optimization = self.expert_optimizer.generate_suggestions(expert_evaluation)
            
            # 步骤5: 存储到Claw记忆
            self.expert_memory.store_evaluation(expert_evaluation, request.chapter_id)
            self.expert_memory.store_optimization(optimization, request.chapter_id)
            
            # 步骤6: 反馈优化（调用iterative-generator-v2）
            optimized_result = self._apply_optimization(base_result, optimization)
            return optimized_result
        
        return base_result
```

### 3.2 插件接口设计

#### 3.2.1 专家插件接口（完全兼容GeneratorPlugin）

```python
from core.plugin_interface import GeneratorPlugin, PluginMetadata, PluginType
from core.models import GenerationRequest, GenerationResult

class ExpertPlugin(GeneratorPlugin):
    """
    专家模式插件接口
    
    设计原则：
    1. 完全继承GeneratorPlugin接口（不破坏现有系统）
    2. 增强不替换（调用现有生成器，不替换）
    3. 可降级（加载失败时回退到默认模式）
    """
    
    def __init__(self, metadata: PluginMetadata):
        """初始化专家插件"""
        super().__init__(metadata)
        
        # 延迟加载的组件
        self._novel_generator = None  # 现有的novel-generator-v3
        self._expert_validator = None  # 专家验证器
        self._expert_optimizer = None  # 专家优化器
        self._expert_memory = None     # Claw记忆集成
        self._local_model = None       # 本地模型辅助
        
        # 专家配置
        self.expert_config = {
            "enable_memory": True,
            "enable_local_model": True,
            "quality_threshold": 0.8,
            "max_iterations": 5
        }
    
    def initialize(self, context: PluginContext) -> None:
        """
        延迟初始化
        
        关键：不在__init__中初始化，避免影响启动速度
        """
        super().initialize(context)
        
        # 延迟加载依赖插件
        self._novel_generator = context.plugin_registry.get_plugin("novel-generator-v3")
        
        if not self._novel_generator:
            raise RuntimeError("依赖插件 novel-generator-v3 未找到")
    
    def generate(self, request: GenerationRequest) -> GenerationResult:
        """
        生成小说内容（专家增强版）
        
        完全兼容GeneratorPlugin接口
        
        核心增强：
        1. 数据源整合（世界观/人设/大纲/风格/知识库/写作技巧）
        2. 强制检查【本章完】标记
        3. 九维度智能评分
        4. 优化建议生成
        5. Claw记忆集成
        
        Args:
            request: 生成请求（与现有接口一致）
            
        Returns:
            GenerationResult: 生成结果（与现有接口一致）
        """
        # 步骤1: 增强请求（整合数据源）
        enhanced_request = self._enhance_request(request)
        
        # 步骤2: 调用现有生成器（不替换）
        base_result = self._novel_generator.generate(enhanced_request)
        
        # 步骤3: 专家评分（含强制检查）
        expert_evaluation = self._evaluate_expert(
            base_result.content,
            enhanced_request
        )
        
        # 步骤4: 检查是否因【本章完】缺失而失败
        if expert_evaluation.total_score == 0.0 and "本章完" in str(expert_evaluation.issues):
            logger.warning("检测到【本章完】标记缺失，自动补充并重新评分")
            
            # 自动补充【本章完】标记
            if not base_result.content.strip().endswith("。"):
                base_result.content = base_result.content.rstrip() + "。\n【本章完】"
            else:
                base_result.content = base_result.content.rstrip() + "\n【本章完】"
            
            # 重新评分
            expert_evaluation = self._evaluate_expert(
                base_result.content,
                enhanced_request
            )
        
        # 步骤5: 如果不达标，生成优化建议
        if expert_evaluation.total_score < self.expert_config["quality_threshold"]:
            optimization = self._generate_optimization(expert_evaluation)
            
            # 步骤6: 存储到记忆
            self._store_to_memory(expert_evaluation, optimization, request)
            
            # 步骤7: 应用优化（重新生成）
            return self._apply_optimization(base_result, optimization)
        
        return base_result
    
    def _enhance_request(self, request: GenerationRequest) -> GenerationRequest:
        """
        增强请求（整合数据源）
        
        这是专家模式的核心增强点：
        将世界观、人设、大纲、风格、知识库、写作技巧
        整合到GenerationRequest中
        """
        # 创建增强请求（不修改原始请求）
        enhanced = GenerationRequest.from_dict(request.to_dict())
        
        # 整合数据源
        if hasattr(request, 'worldview_reference'):
            enhanced.worldview_data = self._load_worldview(request.worldview_reference)
        
        if hasattr(request, 'character_references'):
            enhanced.character_data = self._load_characters(request.character_references)
        
        if hasattr(request, 'outline_reference'):
            enhanced.outline_data = self._load_outline(request.outline_reference)
        
        if hasattr(request, 'style_reference'):
            enhanced.style_data = self._load_style(request.style_reference)
        
        # 新增：知识库和写作技巧
        enhanced.knowledge_base = self._load_knowledge_base()
        enhanced.writing_techniques = self._load_writing_techniques()
        
        return enhanced
    
    # 其他方法在后续章节详细设计
```

## 四、智能评分反馈组件

### 4.1 评分维度设计

#### 4.1.1 九维度评分权重（优化版）

**设计原则**：
- ✅ **继承不冲突** - 前个维度继承EnhancedWeightedValidator的V1.1版本
- ✅ **扩展不破坏** - 新增维度不影响现有评分逻辑
- ✅ **权重可配置** - 支持从YAML配置文件加载
- ✅ **强制标记检查** - 【本章完】标记强制检查，缺失即失败

| 维度 | 权重 | 说明 | 数据源 | 继承关系 |
|------|------|------|--------|---------|
| **章节完整标记** | **一票否决** | **强制检查【本章完】标记** | **章节结尾** | **🆕 强制检查项** |
| 世界观 | 12% | 世界观一致性检查 | data/worldview/ | ✅ 继承自V1.1 |
| 人设 | 19% | 人物性格、对话一致性 | data/characters/ | ✅ 继承自V1.1 |
| 大纲 | 13% | 情节推进符合大纲 | data/outlines/ | ✅ 继承自V1.1 |
| 风格 | 19% | 写作风格匹配度 | data/styles/ + style_learner | ✅ 继承自V1.1 |
| 字数 | 8% | 字数达标率 | 目标字数 | ✅ 继承自V1.1 |
| 知识库 | 8% | 知识点引用质量 | data/knowledge/ | 🆕 **新增维度** |

```python
class ExpertValidator(EnhancedWeightedValidator):
    """
    专家评分器
    
    继承EnhancedWeightedValidator（V1.1版本8维度评分）
    扩展为9维度评分
    
    设计原则：
    1. 继承不破坏 - 调用父类评分方法
    2. 扩展不冲突 - 新增维度不影响现有维度
    3. 降级可用 - 父类失败时使用基础评分
    """
    
    def __init__(self, weights_config_path: str = "config/validator_weights.yaml"):
        """
        初始化专家评分器
        
        Args:
            weights_config_path: 权重配置文件路径
        """
        # 调用父类初始化（加载V1.1版本8维度权重）
        super().__init__(weights_config_path)
        
        # 扩展权重配置（新增维度）
        self._extend_weights()
        
        # 本地模型辅助（延迟初始化）
        self._local_model = None
    
    def _extend_weights(self):
        """
        扩展权重配置
        
        从V1.1版本8维度扩展为V2.0版本9维度
        """
        # 获取V1.1版本的权重
        v11_weights = self.weight_config.to_dict()
        
        # 扩展为V2.0版本权重
        self.expert_weights = {
            # 继承的6个维度
            "世界观": v11_weights["worldview"],
            "人设": v11_weights["character"],
            "大纲": v11_weights["outline"],
            "风格": v11_weights["style"],
            "字数": v11_weights["word_count"],
            "上下文衔接": 0.08,  # 从reverse_feedback (0.11) 调整
            
            # 新增的3个维度
            "知识库": v11_weights["knowledge_reference"],
            "写作技巧": 0.08,  # 全新维度
            "AI感": 0.05,  # 从naturalness (0.10) 调整
        }
        
        # 归一化（确保总和为1.0）
        total = sum(self.expert_weights.values())
        for key in self.expert_weights:
            self.expert_weights[key] /= total
    
    def evaluate(self, content: str, context: Dict[str, Any]) -> ExpertEvaluation:
        """
        九维度评分（含强制检查）
        
        Args:
            content: 生成的内容
            context: 上下文信息
                {
                    "worldview": 世界观数据,
                    "characters": 人物数据,
                    "outline": 大纲数据,
                    "style_profile": 风格数据,
                    "knowledge_base": 知识库数据,
                    "techniques": 写作技巧数据,
                    "previous_chapters": 前文数据
                }
            
        Returns:
            ExpertEvaluation: 评估结果
        """
        scores = {}
        analysis = {}
        
        # ===== 第零步：强制检查【本章完】标记 =====
        # 这是专家模式的核心强制规则，缺失即失败
        if not self._check_chapter_end_marker(content):
            logger.error("【本章完】标记缺失，强制返回失败评分")
            return self._create_failed_evaluation("章节缺少【本章完】标记")
        
        try:
            # ===== 第一步：调用父类评分（V1.1版本6维度） =====
            # 这确保了基础评分的稳定性
            
            # 构建ValidationScores
            base_scores = ValidationScores(
                word_count=self._evaluate_word_count(content, context.get("target_words", 3500)),
                knowledge_reference=self._evaluate_knowledge_reference(content, context.get("knowledge_base", {})),
                outline=self._evaluate_outline(content, context.get("outline", {})),
                style=self._evaluate_style(content, context.get("style_profile", {})),
                character=self._evaluate_character(content, context.get("characters", [])),
                worldview=self._evaluate_worldview(content, context.get("worldview", {})),
                reverse_feedback=self._evaluate_reverse_feedback(content, context.get("previous_chapters", [])),
                naturalness=self._evaluate_naturalness(content)
            )
            
            # 映射到专家评分
            scores["世界观"] = base_scores.worldview
            scores["人设"] = base_scores.character
            scores["大纲"] = base_scores.outline
            scores["风格"] = base_scores.style
            scores["字数"] = base_scores.word_count
            scores["知识库"] = base_scores.knowledge_reference
            
            # ===== 第二步：扩展评分（新增3维度） =====
            
            # 上下文衔接（重新定义）
            scores["上下文衔接"] = self._evaluate_context_continuation(
                content, 
                context.get("previous_chapters", [])
            )
            
            # 写作技巧（全新维度）
            scores["写作技巧"] = self._evaluate_writing_technique(
                content,
                context.get("techniques", {})
            )
            
            # AI感（使用本地模型）
            scores["AI感"] = self._evaluate_ai_sense(content)
            
            # ===== 第三步：生成分析 =====
            analysis = self._generate_analysis(scores, content, context)
            
        except Exception as e:
            # 降级方案：使用基础评分
            logger.error(f"专家评分失败，降级为基础评分: {e}")
            scores = self._fallback_scores()
            analysis = {"error": f"评分异常: {str(e)}"}
        
        # ===== 第四步：计算总分 =====
        total_score = sum(scores[k] * self.expert_weights[k] for k in scores)
        
        # ===== 第五步：识别问题和优势 =====
        issues = self._identify_issues(scores, analysis)
        strengths = self._identify_strengths(scores, analysis)
        
        return ExpertEvaluation(
            total_score=total_score,
            dimension_scores=scores,
            analysis=analysis,
            issues=issues,
            strengths=strengths
        )
```

```python
def _check_chapter_end_marker(self, content: str) -> bool:
    """
    强制检查章节结尾的【本章完】标记
    
    检查规则：
    1. 检查章节最后100个字符
    2. 匹配模式：【本章完】、[本章完]、（本章完）
    3. 缺失即返回False，触发重新生成
    
    Args:
        content: 生成的章节内容
        
    Returns:
        bool: True表示标记存在，False表示缺失
    """
    if not content or len(content.strip()) == 0:
        return False
    
    # 获取最后100个字符
    last_100_chars = content.strip()[-100:]
    
    # 匹配模式列表
    marker_patterns = [
        "【本章完】",
        "[本章完]",
        "（本章完）",
        "(本章完)",
        "本章完"
    ]
    
    # 检查是否包含任一标记
    has_marker = any(pattern in last_100_chars for pattern in marker_patterns)
    
    if not has_marker:
        logger.warning(f"章节结尾未找到【本章完】标记。结尾内容：{last_100_chars}")
    
    return has_marker

def _create_failed_evaluation(self, reason: str) -> ExpertEvaluation:
    """
    创建失败评分（用于强制检查失败时）
    
    Args:
        reason: 失败原因
        
    Returns:
        ExpertEvaluation: 总分为0的评估结果
    """
    return ExpertEvaluation(
        total_score=0.0,
        dimension_scores={
            "世界观": 0.0,
            "人设": 0.0,
            "大纲": 0.0,
            "风格": 0.0,
            "知识库": 0.0,
            "写作技巧": 0.0,
            "字数": 0.0,
            "上下文衔接": 0.0,
            "AI感": 0.0
        },
        analysis={"error": reason},
        issues=[reason, "必须添加【本章完】标记在章节结尾"],
        strengths=[]
    )
```

```python
def _evaluate_worldview(self, content: str, worldview_data: Dict) -> float:
    """
    世界观一致性评分
    
    检查点：
    1. 魔法体系是否一致
    2. 地理设定是否一致
    3. 社会制度是否一致
    4. 时间线是否一致
    """
    score = 0.0
    
    # 提取世界观关键设定
    worldview_rules = worldview_data.get("rules", [])
    worldview_elements = worldview_data.get("elements", [])
    
    # 检查每个世界观元素在内容中的体现
    matched_elements = 0
    for element in worldview_elements:
        if self._check_worldview_element(content, element):
            matched_elements += 1
    
    # 计算匹配率
    if len(worldview_elements) > 0:
        score = matched_elements / len(worldview_elements)
    
    # 检查世界观规则是否违反
    violations = 0
    for rule in worldview_rules:
        if self._check_worldview_violation(content, rule):
            violations += 1
    
    # 违规扣分
    score -= violations * 0.1
    
    return max(0.0, min(1.0, score))
```

```python
class ExpertMemory:

    def __init__(self, vector_store: VectorStore, session_state: SessionState):

            f"evaluation_{chapter_id}",

                "chapter_id": chapter_id,
                "total_score": evaluation.total_score,
                "timestamp": datetime.now().isoformat()
            }],
            ids=[f"eval_{chapter_id}"]
        )

        存储优化建议到记忆系统

        # L1 热记忆
        self.session_state.set(
            f"optimization_{chapter_id}",
            suggestion.to_dict(),
            ttl=3600
        )
        
        # L2 温记忆
        memory_text = self._serialize_optimization(suggestion)
        self.vector_store.add(
            texts=[memory_text],
            metadatas=[{
                "type": "expert_optimization",
                "chapter_id": chapter_id,
                "priority": suggestion.priority,
                "timestamp": datetime.now().isoformat()
            }],
            ids=[f"opt_{chapter_id}"]
        )

        检索相似问题及解决方案

        """
        results = self.vector_store.search(
            query=issue_description,
            filter={"type": "expert_optimization"},
            top_k=top_k
        )
        
        return results
    
    def learn_from_user_feedback(self, feedback: UserFeedback):

            "suggestions_rejected": feedback.rejected_suggestions,
            "user_rating": feedback.user_rating,
            "timestamp": datetime.now().isoformat()
        }
        
        # 存入L2温记忆
        self.vector_store.add(
            texts=[json.dumps(feedback_record, ensure_ascii=False)],
            metadatas=[{
                "type": "user_feedback",
                "chapter_id": feedback.chapter_id,
                "user_rating": feedback.user_rating
            }],
            ids=[f"feedback_{feedback.chapter_id}_{datetime.now().timestamp()}"]
        )
        
        # 触发记忆优化事件
        self._trigger_memory_optimization(feedback)
```

```python
    else:
        return base_suggestion

def _enhance_with_history(self, base_suggestion: OptimizationSuggestion, 
                          history: List[Dict]) -> OptimizationSuggestion:
    """
    融合历史经验增强建议
    """
    # 提取历史中效果好的建议模式
    effective_patterns = []
    for record in history:
        if record.get("suggestions_accepted"):
            effective_patterns.extend(record["suggestions_accepted"])
    
    # 将有效模式融入当前建议
    enhanced_suggestions = {}
    for dimension, suggestion in base_suggestion.dimension_suggestions.items():
        # 检查历史中是否有该维度的成功建议
        matching_patterns = [
            p for p in effective_patterns 
            if dimension in p
        ]
        
        if matching_patterns:
            # 融合历史成功经验
            enhanced_suggestions[dimension] = f"{suggestion}\n\n历史成功经验：{matching_patterns[0]}"
        else:
            enhanced_suggestions[dimension] = suggestion
    
    return OptimizationSuggestion(
        overall_suggestion=base_suggestion.overall_suggestion,
        dimension_suggestions=enhanced_suggestions,
        examples=base_suggestion.examples,
        priority=base_suggestion.priority
    )
```

```python
class ExpertPluginLoader:

    def load_expert_plugin(self, plugin_id: str) -> Optional[GeneratorPlugin]:
        """
        加载专家插件（带降级方案）
        
        降级路径：
        1. 尝试加载专家插件
        2. 失败 -> 尝试加载基础生成器
        3. 失败 -> 使用内存中的生成器
        """

                logger.info(f"专家插件 {plugin_id} 加载成功")

            logger.warning(f"专家插件依赖缺失: {e}")
            
        except Exception as e:
            logger.error(f"专家插件加载失败: {e}")

            default_plugin = self.plugin_registry.get_plugin("novel-generator-v3")
            if default_plugin:
                logger.info("降级为默认生成器")
                return default_plugin
        except Exception as e:
            logger.error(f"默认生成器加载失败: {e}")

        if hasattr(self, '_cached_generator'):
```

```python
class ExpertValidator(EnhancedWeightedValidator):
    """专家评分器（带降级方案）"""
    
    def evaluate(self, content: str, context: Dict[str, Any]) -> ExpertEvaluation:
        """
        九维度评分（带降级方案）
        """
        try:
            # 尝试专家评分
            return self._expert_evaluate(content, context)
            
        except Exception as e:
            logger.error(f"专家评分失败: {e}")
            
            # 降级方案: 使用父类评分（V1.1版本8维度）
            try:
                base_scores = super().validate(content, context)
                return self._convert_to_expert_evaluation(base_scores)
                
            except Exception as e2:
                logger.error(f"基础评分也失败: {e2}")
                
                # 最终降级: 返回默认评分
                return self._fallback_evaluation()
    
    def _fallback_evaluation(self) -> ExpertEvaluation:
        """
        最终降级评分
        
        返回一个默认的评分结果，避免程序崩溃
        """
        return ExpertEvaluation(
            total_score=0.5,
            dimension_scores={
                "世界观": 0.5,
                "人设": 0.5,
                "大纲": 0.5,
                "风格": 0.5,
                "知识库": 0.5,
                "写作技巧": 0.5,
                "字数": 0.5,
                "上下文衔接": 0.5,
                "AI感": 0.5
```

#### 8.1.3 本地模型加载失败

```python
class LocalModelAssistant:
    """本地模型辅助（带降级方案）"""
    
    def initialize(self):
        """
        延迟初始化（带降级方案）
        """
        if self._initialized:
            return
        
        try:
            # 尝试加载本地模型
            from sentence_transformers import SentenceTransformer
            
            self.model = SentenceTransformer(
                'paraphrase-multilingual-MiniLM-L12-v2',
                cache_folder=self.model_cache_path
            )
            self._initialized = True
            logger.info("本地模型加载成功")
            
        except ImportError:
            logger.warning("sentence_transformers未安装，禁用本地模型辅助")
            self._initialized = False
            
        except Exception as e:
            logger.warning(f"本地模型加载失败: {e}")
            self._initialized = False
    
    def calculate_semantic_similarity(self, text1: str, text2: str) -> float:
        """
        计算语义相似度（带降级方案）
        """
        if not self._initialized:
            self.initialize()
        
        if not self.model:
            # 降级方案: 使用简单的字符串匹配
            return self._simple_similarity(text1, text2)
        
        try:
            embeddings = self.model.encode([text1, text2])
            similarity = cosine_similarity([embeddings[0]], [embeddings[1]])[0][0]
            return float(similarity)
            
        except Exception as e:
            logger.warning(f"语义相似度计算失败: {e}")
            # 降级方案
            return self._simple_similarity(text1, text2)
    
    def _simple_similarity(self, text1: str, text2: str) -> float:
        """
        简单的字符串相似度（降级方案）
        """
        # 使用Jaccard相似度
        set1 = set(text1)
        set2 = set(text2)
        
        intersection = len(set1 & set2)
        union = len(set1 | set2)
        
        return intersection / union if union > 0 else 0.0
```
