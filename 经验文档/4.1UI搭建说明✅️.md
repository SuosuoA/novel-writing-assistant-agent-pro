#### 4. 设计原则

**题材 vs 写作技巧**:
- **题材**：小说分类（玄幻/科幻/言情等），14个
- **写作技巧**：创作技法（叙事/描写/修辞等），65项
- **独立选择器**：写作技巧有独立的双下拉框选择器，不应混入题材分类

**数据一致性**:
- 开始创作界面：题材不包含写作技巧
- 其他页面：保留写作技巧选项（AI生成知识/知识库搜索）

**无边框实现**：

```python
def make_frameless(self, keep_resize_border: bool = True):
    """设置完全无边框窗口"""
    # 1. 通过Windows API移除边框样式
    style = self.user32.GetWindowLongW(self.hwnd, self.GWL_STYLE)
    style &= ~(self.WS_CAPTION | self.WS_SYSMENU | self.WS_BORDER)
    self.user32.SetWindowLongW(self.hwnd, self.GWL_STYLE, style)
    
    # 2. 强制重绘窗口
    self.user32.SetWindowPos(self.hwnd, None, 0, 0, 0, 0, 0x27)
    
    # 3. Tkinter层完全移除标题栏
    self.root.overrideredirect(True)
```

def _setup_system_tray(self):
    """设置系统托盘图标"""
    # 加载图标
    icon_image = Image.open("icon.ico")
    
    # 创建托盘菜单
    menu = pystray.Menu(
        pystray.MenuItem("显示窗口", self._show_window),

## 三、对话框滚动支持

### 3.1 问题

对话框高度700px，内容过多时底部按钮不可见，且无滚动条。

### 3.2 解决方案

```python
# 窗口尺寸
dialog.geometry("750x800")  # 高度从700增加到800

# Canvas + Scrollbar滚动支持
canvas = tk.Canvas(dialog, highlightthickness=0)
scrollbar = ttk.Scrollbar(dialog, orient="vertical", command=canvas.yview)
scrollable_frame = ttk.Frame(canvas, padding=20)

scrollable_frame.bind(
    "<Configure>",
    lambda e: canvas.configure(scrollregion=canvas.bbox("all"))
)

canvas.create_window((0, 0), window=scrollable_frame, anchor="nw")
canvas.configure(yscrollcommand=scrollbar.set)

# 绑定鼠标滚轮
def _on_mousewheel(event):
    canvas.yview_scroll(int(-1*(event.delta/120)), "units")
canvas.bind_all("<MouseWheel>", _on_mousewheel)

# 对话框销毁时解绑
def _on_destroy(event):
    canvas.unbind_all("<MouseWheel>")
dialog.bind("<Destroy>", _on_destroy)
```
