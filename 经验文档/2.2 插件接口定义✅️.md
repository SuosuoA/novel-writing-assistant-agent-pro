## 一、接口概述

### 1.1 设计原则

插件系统遵循以下核心原则：

| 原则 | 说明 |
|------|------|
| **最小化接口** | 只定义必要的方法，避免过度设计 |
| **向后兼容** | 新增方法使用默认实现，不破坏现有插件 |
| **显式依赖** | 插件依赖在清单中明确声明 |
| **清晰边界** | 插件不直接访问核心内部状态 |
| **专用接口** | 不包含通用execute()，使用具体接口方法 |

### 1.2 插件类型体系

```
BasePlugin (基础插件)
    ├── ProtocolPlugin (协议插件) - 导入导出格式转换
    ├── AIPlugin (AI插件) - LLM调用封装
    ├── StoragePlugin (存储插件) - 数据持久化
    ├── AnalyzerPlugin (分析器插件)
    │   ├── OutlineParserPlugin (大纲解析)
    │   ├── StyleLearnerPlugin (风格学习)
    │   ├── CharacterManagerPlugin (人物管理)
    │   └── WorldviewParserPlugin (世界观解析)
    ├── GeneratorPlugin (生成器插件)
    │   ├── ContextBuilderPlugin (上下文构建)
    │   ├── NovelGeneratorPlugin (小说生成)
    │   ├── ContinuationGeneratorPlugin (续写生成) ✅ 新增
    │   └── QuickCreatorPlugin (快捷创作) ✅ 新增
    ├── ValidatorPlugin (验证器插件)
    │   ├── WeightedValidatorPlugin (加权验证)
    │   └── KnowledgeValidatorPlugin (知识验证) ✅ 新增
    └── ToolPlugin (工具插件)
        ├── HotRankingPlugin (热榜功能)
        ├── ReverseFeedbackAnalyzerPlugin (逆向反馈分析) ✅ 新增
        └── KnowledgeVerifierPlugin (知识库验证器) ✅ V2.3新增
```

## 二、PluginMetadata元数据

### 2.1 数据类定义

```python
@dataclass
class PluginMetadata:
    """插件元数据"""
    
    # 必需字段
    id: str                    # 唯一标识符（如：outline-parser-v3）
    name: str                  # 显示名称
    version: str               # 版本号（语义化版本）
    description: str           # 描述
    author: str                # 作者
    plugin_type: PluginType    # 插件类型
    
    # 可选字段
    api_version: str = "1.0"          # API版本
    priority: int = 100               # 加载优先级（越小越先）
    enabled: bool = True              # 是否启用
    dependencies: List[str] = []      # 依赖插件ID列表
    conflicts: List[str] = []         # 冲突插件ID列表
    permissions: List[str] = []       # 所需权限
    min_platform_version: str = "6.0.0"  # 最低平台版本
    entry_class: str = ""             # 入口类名
```

### 2.2 字段说明

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `id` | str | ✅ | 唯一标识符，格式：`^[a-z0-9]+(-[a-z0-9]+)*$` |
| `name` | str | ✅ | 显示名称，最大128字符 |
| `version` | str | ✅ | 语义化版本，格式：`X.Y.Z` |
| `description` | str | ✅ | 功能描述，最大512字符 |
| `author` | str | ✅ | 作者名 |
| `plugin_type` | Enum | ✅ | 插件类型（见2.3节） |
| `api_version` | str | ❌ | API版本，默认"1.0" |
| `priority` | int | ❌ | 加载优先级1-1000，默认100 |
| `enabled` | bool | ❌ | 是否启用，默认True |
| `dependencies` | List[str] | ❌ | 依赖插件ID列表 |
| `conflicts` | List[str] | ❌ | 冲突插件ID列表 |
| `permissions` | List[str] | ❌ | 所需权限列表 |
| `min_platform_version` | str | ❌ | 最低平台版本，默认"6.0.0" |
| `entry_class` | str | ❌ | 入口类名 |

## 四、BasePlugin基础接口

### 4.1 接口定义

```python
class BasePlugin(ABC):
    """插件基类"""
    
    def __init__(self, metadata: PluginMetadata):
        """初始化插件"""
        self.metadata = metadata
        self._context: Optional[PluginContext] = None
        self._cleanup_done = False
        self._state: PluginState = PluginState.LOADED
    
    @classmethod
    @abstractmethod
    def get_metadata(cls) -> PluginMetadata:
        """获取插件元数据（类方法）"""
        pass
    
    @abstractmethod
    def initialize(self, context: PluginContext) -> bool:
        """初始化插件"""
        pass
    
    def shutdown(self) -> bool:
        """关闭插件 - 优雅关闭，释放资源"""
        return True
    
    def cleanup(self) -> bool:
        """强制资源回收 - 与shutdown区分"""
        self._cleanup_done = True
        return True
    
    @property
    def state(self) -> PluginState:
        """获取插件状态"""
        return self._state
    
    @property
    def is_initialized(self) -> bool:
        """是否已初始化"""
        return self._context is not None
```

### 4.2 生命周期状态

```python
class PluginState(Enum):
    """插件状态枚举 - 5状态模型"""
    UNLOADED = "unloaded"  # 未加载（初始状态）
    LOADED = "loaded"      # 已加载（初始化完成）
    ACTIVE = "active"      # 已激活（可执行）
    ERROR = "error"        # 错误状态
    UNLOADING = "unloading"  # 卸载中
```

**状态转换图**：
```
UNLOADED → LOADED       (register成功)
LOADED → ACTIVE         (activate成功)
ACTIVE → LOADED         (deactivate成功)
LOADED → ERROR          (initialize异常)
ACTIVE → ERROR          (execute连续失败)
ERROR → LOADED          (recover成功)
LOADED → UNLOADING      (卸载请求)
ACTIVE → UNLOADING      (卸载请求，需先deactivate)
UNLOADING → UNLOADED    (shutdown+cleanup完成)
```

**重要变更**：
- 新增 `UNLOADED` 状态（初始状态）
- `state` 属性改为只读，通过 `_set_state()` 内部方法修改

### 4.3 重要说明

**BasePlugin不包含execute()方法**：
- ❌ 不使用：`plugin.execute(*args, **kwargs)`
- ✅ 使用专用接口：
  - `AnalyzerPlugin.analyze()`
  - `GeneratorPlugin.generate()`
  - `ValidatorPlugin.validate()`

### 5.1 AnalyzerPlugin（分析器插件）

```python
class AnalyzerPlugin(BasePlugin):
    """分析器插件接口"""
    
    @abstractmethod
    def analyze(self, content: str, 
                options: Optional[Dict[str, Any]] = None) -> Dict[str, Any]:
        """分析内容并返回结果"""
        pass
    
    @abstractmethod
    def get_supported_formats(self) -> List[str]:
        """获取支持的输入格式"""
        pass
    
    @abstractmethod
    def get_analysis_types(self) -> List[str]:
        """获取支持的分析类型"""
        pass
```

**适用场景**：

### 5.2 GeneratorPlugin（生成器插件）

```python
class GeneratorPlugin(BasePlugin):
    """生成器插件接口"""
    
    @abstractmethod
    def generate(self, request: GenerationRequest) -> GenerationResult:
        """生成内容"""
        pass
    
    @abstractmethod
    def validate_request(self, request: GenerationRequest) -> tuple[bool, List[str]]:
        """验证请求是否有效"""
        pass
    
    @abstractmethod
    def get_generation_options(self) -> Dict[str, Any]:

        pass
```

**ValidationScores模型**：

```python
class ValidationScores(BaseModel):
    """V1.7版本 - 8维度评分模型"""
    model_config = ConfigDict(frozen=False)
    
    # 8个评分维度
    word_count_score: float = Field(0.0, ge=0, le=1, description="字数评分（8%）")
    knowledge_ref_score: float = Field(0.0, ge=0, le=1, description="知识点引用评分（8%）")
    outline_score: float = Field(0.0, ge=0, le=1, description="大纲评分（13%）")
    style_score: float = Field(0.0, ge=0, le=1, description="风格评分（19%）")
    character_score: float = Field(0.0, ge=0, le=1, description="人设评分（19%）")
    worldview_score: float = Field(0.0, ge=0, le=1, description="世界观评分（12%）")
    reverse_feedback_score: float = Field(0.0, ge=0, le=1, description="逆向反馈评分（11%）")
    naturalness_score: float = Field(0.0, ge=0, le=1, description="自然度评分（10%）")
    
    total_score: float = Field(0.0, ge=0, le=1, description="总分（加权计算）")
    has_chapter_end: bool = Field(False, description="是否包含章节结束标记")
    
    def calculate_total(self) -> float:
        """8维度加权计算（V1.7版本权重）"""
        self.total_score = (
            self.word_count_score * 0.08 +
            self.knowledge_ref_score * 0.08 +
            self.outline_score * 0.13 +
            self.style_score * 0.19 +
            self.character_score * 0.19 +
            self.worldview_score * 0.12 +
            self.reverse_feedback_score * 0.11 +
            self.naturalness_score * 0.10
        )
        return self.total_score
```

**适用场景**：
- 加权验证（WeightedValidatorPlugin）

## 八、使用示例

### 8.1 完整插件实现（大纲解析器）

```python
# plugins/outline-parser-v3/plugin.py
from core.plugin_interface import (
    AnalyzerPlugin, 
    PluginMetadata, 
    PluginType,
    PluginContext
)
from typing import Dict, Any, List, Optional

class OutlineParserPlugin(AnalyzerPlugin):
    """大纲解析器插件 - V5核心模块迁移示例"""
    
    def __init__(self):
        metadata = PluginMetadata(
            id="outline-parser-v3",
            name="大纲解析器 V3",
            version="3.0.0",
            description="基于LangChain的智能大纲解析",
            author="项目组",
            plugin_type=PluginType.ANALYZER
        )
        super().__init__(metadata)
    
    @classmethod
    def get_metadata(cls) -> PluginMetadata:
        return PluginMetadata(
            id="outline-parser-v3",
            name="大纲解析器 V3",
            version="3.0.0",
            description="基于LangChain的智能大纲解析",
            author="项目组",
            plugin_type=PluginType.ANALYZER
        )
    
    def initialize(self, context: PluginContext) -> bool:
        if not super().initialize(context):
            return False
        
        # 订阅事件
        self._context.event_bus.subscribe(
            "outline.parse_requested",
            self._on_parse_requested
        )
        
        return True
    
    def analyze(self, content: str, 
                options: Optional[Dict[str, Any]] = None) -> Dict[str, Any]:
        """解析大纲内容
        
        Args:
            content: 大纲文本内容
            options: 解析选项（可选）
            
        Returns:
            解析结果字典，包含章节列表
        """
        # 实现解析逻辑（示例）
        chapters = self._extract_chapters(content)
        return {
            "chapters": chapters,
            "total_count": len(chapters),
            "format": options.get("format", "auto") if options else "auto"
        }
    
    def get_supported_formats(self) -> List[str]:
        """获取支持的输入格式"""
        return ["txt", "json", "yaml", "md"]
    
    def get_analysis_types(self) -> List[str]:
        """获取支持的分析类型"""
        return ["outline", "structure", "chapters"]
    
    def _extract_chapters(self, content: str) -> List[Dict]:
        """提取章节列表（内部方法）"""
        # 实现提取逻辑（简化示例）
        chapters = []
        lines = content.split("\n")
        for line in lines:
            if line.startswith("第") and "章" in line:
                chapters.append({"title": line.strip(), "content": ""})
        return chapters
    
    def _on_parse_requested(self, event):
        """处理解析请求事件"""
        result = self.analyze(event.data.get("content"))
        self._context.event_bus.publish(
            "outline.parse_completed",
            {"result": result, "request_id": event.data.get("request_id")}
        )
    
    def shutdown(self) -> bool:
        """优雅关闭"""
        # 取消事件订阅
        if self._context:
            self._context.event_bus.unsubscribe("outline.parse_requested")
        return super().shutdown()
```

```python
            生成结果（包含内容和评分）
        """
        content = ""
        iteration_count = 0
        
        # 迭代生成逻辑（简化示例）
        for i in range(request.max_iterations):
            iteration_count = i + 1
            # 调用LLM生成...
            # 验证评分...
            # 如果评分达标则break
            
        return GenerationResult(
            request_id=request.request_id,
            content=content,
            word_count=len(content),
            iteration_count=iteration_count,
            validation_scores=None  # 由验证器填充
        )
    
    def validate_request(self, request: GenerationRequest) -> Tuple[bool, List[str]]:
        """验证请求是否有效"""
        errors = []
        
        if not request.title:
            errors.append("章节标题不能为空")
        if request.word_count < 500 or request.word_count > 10000:
            errors.append(f"字数范围应为500-10000，当前为{request.word_count}")
        if not request.outline:
            errors.append("章节大纲不能为空")
            
        return len(errors) == 0, errors
    
    def get_generation_options(self) -> Dict[str, Any]:
        """获取生成选项定义"""
        return {
            "word_count": {
                "type": "integer",
```

### 10.2 订阅示例

```python
def initialize(self, context: PluginContext) -> bool:
    self._context = context
    
    # 订阅事件
    self._context.event_bus.subscribe(
        "generation.started",
        self._on_generation_started,
        handler_id=f"{self.metadata.id}.gen_started"
    )
    
    return True

def _on_generation_started(self, event):
    """事件处理函数"""
    # Tkinter线程安全：使用after()调度到主线程
    def update_ui():
        print(f"生成开始: {event.data.get('chapter_title')}")
    
    # 如果需要操作UI
    if hasattr(self._context, 'root'):
        self._context.root.after(0, update_ui)
    else:
        update_ui()

def shutdown(self) -> bool:
    """取消订阅"""
    self._context.event_bus.unsubscribe(
        f"{self.metadata.id}.gen_started"
    )
    return True
```
