## 三、技术要点

### 3.1 核心接口设计

**EventBus线程安全**:
- 读拷贝快照模式: 在锁内拷贝handlers快照,避免长时间持锁
- ThreadPoolExecutor异步执行: 不阻塞调用线程
- 熔断器集成: FAILURE_THRESHOLD=5, RECOVERY_TIMEOUT=60s

**PluginRegistry状态机**:
```
UNLOADED → LOADED → ACTIVE → UNLOADING → (移除)
              ↓         ↓
            ERROR ←←←←←←
```

**ServiceLocator依赖管理**:
- 循环依赖检测: 通过`dependencies`属性声明依赖
- 生命周期管理: `initialize_all()` / `dispose_all()`
- 作用域支持: SINGLETON / TRANSIENT / SCOPED

### 3.2 插件开发规范

**plugin.json必需字段**:
- id: 唯一标识符,格式`^[a-z0-9]+(-[a-z0-9]+)*$`
- name: 显示名称,最大128字符
- version: 语义化版本,X.Y.Z格式
- plugin_type: 插件类型枚举
- entry_class: 入口类名

**插件类必需实现**:
- `get_metadata()`: 类方法,返回PluginMetadata
- `initialize(context)`: 初始化插件
- `shutdown()`: 优雅关闭(可选)
- 专用接口方法(analyze/generate/validate等)

### 3.3 长期记忆接口

**SESSION-STATE管理器**:
```python
# 写入状态(WAL协议)
session_manager.write_state({
    "timestamp": datetime.now().isoformat(),
    "active_task": "章节创作",
    "pending_data": {...}
})

# 恢复状态
state = session_manager.recover_state()
```

**向量库接口**:
```python
# 添加章节向量
vector_store.add_chapter(
    chapter_id="chapter_003",
    content="第三章内容...",
    metadata={...}
)

# 召回相似上下文
similar = vector_store.recall_similar_context(
    query="主角林峰回到故乡",
    top_k=10
)
```

### 3.4 知识库接口

**知识点Schema**:
```python
class KnowledgePoint(BaseModel):
    knowledge_id: str
    category: str          # 科幻/玄幻/历史
    domain: str            # 物理/化学/生物等
    title: str
    content: str
    keywords: List[str]
    references: List[str]
```

**检索接口**:
```python
# 向量检索
results = retriever.retrieve(
    query="飞船穿越虫洞",
    category="科幻",
    domain="物理学",
    top_k=10,
    min_score=0.7
)
```
