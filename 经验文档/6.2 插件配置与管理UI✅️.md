## 一、功能概述

### 1.1 功能列表

| 功能 | 说明 |
|------|------|
| 插件列表显示 | 显示已安装插件名称、版本、状态、类型、保护状态 |
| 启用/禁用插件 | 通过PluginRegistry动态启用或禁用插件 |
| 插件配置 | 通过ConfigManager读取/保存插件配置 |
| V5保护模块标识 | 核心模块禁止禁用/卸载/重新加载 |
| 插件详情查看 | 显示插件完整信息（依赖、作者、描述等） |

### 1.2 设计原则

1. **线程安全**：所有UI操作通过主线程执行
2. **状态同步**：操作后自动刷新列表
3. **保护机制**：V5核心模块禁止危险操作
4. **配置持久化**：插件配置通过ConfigManager保存

## 四、启用/禁用功能

### 4.1 启用插件

```python
def _on_enable_plugin(self) -> None:
    """启用插件 - 调用PluginRegistry.activate()"""
    plugin_data = self._get_selected_plugin()
    plugin_id = plugin_data["id"]
    
    # 1. 检查保护状态
    if plugin_data.get("is_protected"):
        messagebox.showwarning("警告", "V5核心保护模块，无法操作")
        return
    
    # 2. 检查当前状态
    if plugin_data.get("state") == "active":
        messagebox.showinfo("提示", "已经处于启用状态")
        return
    
    # 3. 调用PluginRegistry
    success = self._services.registry.activate(plugin_id)
    
    # 4. 刷新列表
    if success:
        self._load_plugins()
        messagebox.showinfo("成功", f"插件 '{plugin_id}' 已成功启用")
```

### 4.2 禁用插件

```python
def _on_disable_plugin(self) -> None:
    """禁用插件 - 调用PluginRegistry.deactivate()"""
    plugin_data = self._get_selected_plugin()
    plugin_id = plugin_data["id"]
    
    # 1. 检查保护状态
    if plugin_data.get("is_protected"):
        messagebox.showwarning("警告", "V5核心保护模块，禁止禁用")
        return
    
    # 2. 检查当前状态
    if plugin_data.get("state") != "active":
        messagebox.showinfo("提示", "当前未启用")
        return
    
    # 3. 确认对话框
    if not messagebox.askyesno("确认", f"确定要禁用插件 '{plugin_id}' 吗？"):
        return
    
    # 4. 调用PluginRegistry
    success = self._services.registry.deactivate(plugin_id)
    
    # 5. 刷新列表
    if success:
        self._load_plugins()
```

```python
def _show_plugin_config_dialog(self, plugin_id: str, plugin_name: str, plugin_data: Dict):
    """显示插件配置对话框"""
    
    # 从ConfigManager加载配置
    plugin_config = {}
    if self._services.config:
        config_keys = self._services.config.list_keys(f"plugins.{plugin_id}")
        for key in config_keys:
            value = self._services.config.get(key)
            plugin_config[key] = value
    
    # 如果没有配置，显示默认配置选项
    if not plugin_config:
        default_configs = self._get_default_plugin_config(plugin_id, plugin_type)
        # ... 创建配置控件
```

```python
def _create_plugins_page(self) -> tk.Frame:

    frame = ttk.Frame(self._content_frame, style="TFrame")
    
    # 滚动区域
    canvas = tk.Canvas(frame, bg=GlassTheme.GLASS_BG)
    scrollbar = ttk.Scrollbar(frame, orient="vertical", command=canvas.yview)
    scrollable_frame = ttk.Frame(canvas, style="TFrame")
    
    # 插件列表 Treeview
    columns = ("name", "version", "status", "type", "protected")
    self._plugin_tree = ttk.Treeview(list_frame, columns=columns, show="headings")
```
