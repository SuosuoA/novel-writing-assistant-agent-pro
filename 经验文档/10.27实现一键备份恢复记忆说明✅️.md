## 一、任务概述

实现记忆备份/恢复命令行工具，支持一键备份所有记忆层级（L1-L4）和选择性备份恢复功能。

---

## 二、核心功能

### 2.1 备份层级

| 层级 | 名称 | 备份内容 | 说明 |
|------|------|----------|------|
| L1 | 热记忆 | session-state.md, wal.json | 当前会话状态、WAL日志 |
| L2 | 温记忆 | vector_store/, chapter_encoding_history.json | 向量数据库、章节编码历史 |
| L3 | 冷记忆 | .git/refs/notes | Git-Notes分支感知历史 |
| L4 | 精选档案 | .workbuddy/memory/ | MEMORY.md、日志文档 |
| config | 配置文件 | config.yaml, settings.local.json | 项目配置 |

### 2.2 功能特性

1. **完整备份**: 备份所有记忆层级（L1-L4 + config）
2. **选择性备份**: 支持 `-l` 参数指定备份层级
3. **压缩备份**: 默认使用 tar.gz 格式压缩
4. **验证恢复**: 恢复前验证备份完整性和校验和
5. **自动备份**: 恢复前自动创建当前状态备份

---

## 三、命令行接口

### 3.1 备份命令

```bash
# 完整备份
python tools/memory_backup.py backup

# 选择性备份（只备份L4）
python tools/memory_backup.py backup -l L4

# 指定输出目录和描述
python tools/memory_backup.py backup -o E:\backups -d "发布前备份"

# 不压缩
python tools/memory_backup.py backup --no-compress
```

### 3.2 恢复命令

```bash
# 恢复备份
python tools/memory_backup.py restore -i memory_backups/backup_20260326.tar.gz

# 选择性恢复（只恢复L4）
python tools/memory_backup.py restore -i backup.tar.gz -l L4

# 不验证直接恢复
python tools/memory_backup.py restore -i backup.tar.gz --no-verify

# 恢复前不创建备份
python tools/memory_backup.py restore -i backup.tar.gz --no-backup
```

### 3.3 列出备份

```bash
# 列出所有备份
python tools/memory_backup.py list

# 指定备份目录
python tools/memory_backup.py list -o E:\backups
```

## 四、核心类

### 4.1 数据模型

| 类 | 说明 |
|---|---|
| `BackupManifest` | 备份清单（版本、创建时间、层级、校验和等） |
| `BackupResult` | 备份结果（成功/失败、路径、文件数、大小） |
| `RestoreResult` | 恢复结果（成功/失败、恢复层级、文件数、警告） |

### 4.2 主要方法

| 方法 | 说明 |
|------|------|
| `backup()` | 执行备份 |
| `restore()` | 恢复备份 |
| `list_backups()` | 列出所有备份 |
| `_calculate_checksum()` | 计算备份校验和 |
| `_verify_backup()` | 验证备份完整性 |
| `_compress_backup()` | 压缩备份 |
| `_decompress_backup()` | 解压备份 |

## 七、与GUI集成

GUI中的备份/恢复按钮（`_on_backup_memory` / `_on_restore_memory`）已实现类似功能，可通过以下方式集成此工具：

```python
# 方式1：直接调用工具类
from tools.memory_backup import MemoryBackupTool

tool = MemoryBackupTool(workspace_root)
result = tool.backup(description="手动备份")

# 方式2：通过subprocess调用命令行
import subprocess
result = subprocess.run([
    "python", "tools/memory_backup.py", "backup", 
    "-d", "手动备份"
], capture_output=True, text=True)
```

## 八、注意事项

1. **恢复前备份**: 默认在恢复前创建当前状态备份，防止误操作
2. **校验和验证**: 恢复时验证备份完整性，确保数据正确
3. **压缩格式**: 使用 tar.gz 格式，兼容性好、压缩率高
4. **层级选择**: 可选择性备份/恢复特定层级，灵活控制
5. **备份目录**: 默认存储在 `memory_backups/` 目录
