## 九、使用方式

### 9.1 通过插件管理器加载

```python
from core.plugin_registry import PluginRegistry

registry = PluginRegistry()
registry.load_plugin("outline-parser-v3")

result = registry.execute("outline-parser-v3", content="大纲文本...")
```

### 9.2 直接导入使用

```python
from plugins.outline_parser_v3.plugin import OutlineParserPlugin

plugin = OutlineParserPlugin()
plugin.initialize()
result = plugin.analyze(content="大纲文本...")
```

### 9.3 UI回调示例

```python

def on_new_outline(name, novel_type, target_words, description):
    plugin = OutlineParserPlugin()

    if result["success"]:
        # 更新UI列表
        pass

# 风格学习 - 创建模板回调
def on_create_style_template(name, description, dimensions):
    plugin = StyleLearnerPlugin()
    result = plugin.create_style_template(name, description, dimensions)
    if result["success"]:
        # 添加到风格档案库
        pass
```
