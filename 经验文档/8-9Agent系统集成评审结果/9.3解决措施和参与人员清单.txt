### P0-1: Agent任务payload输入验证

**修复位置**: `agents/master_agent.py:299-335` (submit_task方法)

**修复方案**:

1. **创建验证模型**（新文件 `agents/validation.py`）
```python
from pydantic import BaseModel, Field, field_validator
import re

class TaskPayload(BaseModel):
    """任务payload验证模型"""
    prompt: str = Field(..., min_length=1, max_length=10000)
    options: Dict[str, Any] = Field(default_factory=dict)
    
    @field_validator("prompt")
    @classmethod
    def validate_prompt(cls, v: str) -> str:
        # 检查危险模式
        dangerous_patterns = [
            r"__import__",
            r"eval\s*\(",
            r"exec\s*\(",
            r"DROP\s+TABLE",
            r"\.\./",  # 路径遍历
            r"os\.system",
            r"subprocess\.",
        ]
        for pattern in dangerous_patterns:
            if re.search(pattern, v, re.IGNORECASE):
                raise ValueError(f"Dangerous pattern detected: {pattern}")
        
        # 移除危险字符
        v = re.sub(r'[<>\"\'\\]', '', v)
        return v.strip()
```

2. **修改submit_task方法**

**修复位置**: `agents/master_agent.py:240-297` (_handle_task_failure方法)

**修复方案**:

1. **定义不可重试异常类型**

```python
NON_RETRYABLE_ERRORS = (
    ValueError,        # 输入验证错误
    PermissionError,   # 权限错误
    ImportError,       # 模块加载错误
    SyntaxError,       # 语法错误
    TypeError,         # 类型错误
)
```

2. **修改_handle_task_failure方法**
```python
from agents.constants import NON_RETRYABLE_ERRORS

def _handle_task_failure(self, task: AgentTask, error: str, constraints: Any) -> None:
    # ✅ 检查失败原因
    error_type = type(task.last_exception) if hasattr(task, 'last_exception') else None
    
    if error_type and issubclass(error_type, NON_RETRYABLE_ERRORS):
        logger.warning(f"任务 {task.task_id} 因不可重试错误失败: {error_type.__name__}")
        self._dependency_state.mark_failed(task.task_id)
        # 直接使用降级策略
        if constraints:
            self._use_fallback_strategy(task, constraints)
        return
    
    # ✅ 添加指数退避延迟
    if task.can_retry:
        import time
        delay = min(2 ** task.retry_count, 60)  # 最大60秒
        logger.info(f"任务 {task.task_id} 等待 {delay}秒后重试")
        time.sleep(delay)
        
        task.retry_count += 1
        self._task_queue.push(task)
    else:
        # ... 原有降级逻辑
```

**预期效果**: 避免无限重试导致的资源耗尽

**验证方法**: 模拟插件崩溃，验证重试行为

### P1-1: 嵌套任务超时累积

**修复位置**: `agents/priority.py` (AgentTask类)

**修复方案**:

1. **扩展AgentTask类**
```python
@dataclass
class AgentTask:
    # ... 原有字段
    global_timeout: int = 60  # 全局超时限制（秒）
    started_at: Optional[datetime] = None
    
    @property
    def global_timeout_exceeded(self) -> bool:
        """检查是否超过全局超时"""
        if not self.started_at:
            return False
        elapsed = (datetime.now(timezone.utc) - self.started_at).total_seconds()
        return elapsed > self.global_timeout
```

2. **修改_execute_task_sync方法**
```python
def _execute_task_sync(self, task: AgentTask) -> None:
    # ✅ 检查全局超时
    if task.global_timeout_exceeded:
        logger.warning(f"任务 {task.task_id} 已超过全局超时，跳过执行")
        self._dependency_state.mark_failed(task.task_id)
        self._event_bus.publish(
            "agent.task.global_timeout",
            {"task_id": task.task_id, "elapsed": elapsed},
            source="MasterAgent"
        )
        return
    
    # ... 原有逻辑
```

**预期效果**: 任务链总时间可控

**验证方法**: 测试任务链执行时间

### P1-4: Agent权限模型

**修复位置**: 新文件 `agents/permissions.py`

1. **定义权限枚举**

```python
from enum import Enum

class AgentPermission(Enum):
    """Agent权限枚举"""
    READ = "read"          # 读取权限
    WRITE = "write"        # 写入权限
    EXECUTE = "execute"    # 执行权限
    ADMIN = "admin"        # 管理权限

# Agent默认权限配置
DEFAULT_PERMISSIONS = {
    "thinker": [AgentPermission.READ, AgentPermission.EXECUTE],
    "optimizer": [AgentPermission.READ, AgentPermission.WRITE, AgentPermission.EXECUTE],
    "validator": [AgentPermission.READ, AgentPermission.EXECUTE],
    "planner": [AgentPermission.READ, AgentPermission.WRITE, AgentPermission.EXECUTE],
}
```

2. **在AgentMetadata中添加permissions字段**
```python
@dataclass
class AgentMetadata:
    # ... 原有字段
    permissions: List[AgentPermission] = field(default_factory=list)
```

3. **在MasterAgent中添加权限检查**
```python
def _check_permission(self, agent: BaseAgent, required_permission: AgentPermission) -> bool:
    """检查Agent权限"""
    if required_permission not in agent.metadata.permissions:
        logger.warning(
            f"Agent {agent.agent_type} 无 {required_permission.value} 权限"
        )
        self._event_bus.publish(
            "agent.permission.denied",
            {
                "agent_type": agent.agent_type,
                "required_permission": required_permission.value,
            },
            source="MasterAgent"
        )
        return False
    return True
```

**预期效果**: 防止权限提升和越权执行

**验证方法**: 测试Agent权限检查
