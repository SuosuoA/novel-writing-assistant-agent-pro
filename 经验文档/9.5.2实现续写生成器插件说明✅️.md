## 一、概述

### 1.1 功能定位

续写生成器插件（ContinuationGeneratorPlugin）是Agent Pro项目的核心生成器插件之一，提供智能小说续写功能。该插件继承自 `ContinuationPlugin` 接口，实现了完整的续写工作流。

### 1.2 核心能力

| 能力 | 描述 |
|------|------|
| 单次续写 | 基于起始文本和上下文生成续写内容 |
| 多版本生成 | 生成多个不同风格的版本供用户选择 |
| 选择最佳版本 | 通过多维度评分选择最优版本 |
| 流式生成 | 实时输出续写内容 |
| 重新生成 | 支持创意/保守/聚焦等不同模式 |
| 效果预估 | 预测续写效果和资源消耗 |

```python
def generate_continuation(
    self, 
    request: ContinuationRequest
) -> ContinuationResult:
    """执行续写 - 核心方法"""
    # 1. 构建Prompt（整合大纲、人设、世界观、风格、前文）
    prompt = self._build_continuation_prompt(request)
    
    # 2. 计算max_tokens
    max_tokens = self._calculate_max_tokens(request.word_count)
    
    # 3. 调用大模型API
    generated_text = self._call_llm_api(
        prompt=prompt,
        max_tokens=max_tokens,
        temperature=request.temperature
    )
    
    # 4. 后处理（字数控制、格式清理）
    processed_text = self._post_process(generated_text, request)
    
    # 5. 构建结果
    return ContinuationResult(...)
```

### 3.2 多版本生成

```python
def generate_multiple_versions(
    self,
    request: ContinuationRequest,
    num_versions: int = 3,
    temperatures: Optional[List[float]] = None
) -> List[ContinuationResult]:
    """生成多个版本的续写"""
    # 默认温度分布：保守(0.6) / 平衡(0.8) / 创意(1.0)
    if temperatures is None:
        temperatures = [0.6, 0.8, 1.0]
    
    results = []
    for temp in temperatures[:num_versions]:
        version_request = ContinuationRequest(
            ...,
            temperature=temp
        )
        result = self.generate_continuation(version_request)
        results.append(result)
    
    return results
```

### 3.3 选择最佳版本

**评分维度**:

| 维度 | 权重 | 说明 |
|------|------|------|
| 连贯性 | 30% | 续写与起始文本的衔接自然度 |
| 风格匹配 | 25% | 与原始风格的匹配程度 |
| 字数达标 | 20% | 是否达到目标字数 |
| 创意性 | 15% | 词汇多样性、句式变化 |
| 可读性 | 10% | 段落结构、标点使用 |

```python
def select_best_version(
    self,
    versions: List[ContinuationResult],
    request: ContinuationRequest,
    criteria: Optional[Dict[str, float]] = None
) -> tuple[ContinuationResult, int, Dict[str, Any]]:
    """选择最佳版本"""
    # 评估每个版本
    for version in versions:
        # 1. 连贯性评分（来自metadata）
        coherence = version.metadata.coherence_score
```

### 3.4 创意性评估

```python
def _evaluate_creativity(self, text: str) -> float:
    """评估文本创意性"""
    # 1. 词汇多样性（去重词汇占比）
    words = list(jieba.cut(text))
    diversity = len(set(words)) / len(words)
    
    # 2. 句式变化（不同长度句子的方差）
    sentences = text.split('。')
    variance = calculate_length_variance(sentences)
    sentence_variety = min(1.0, variance / 100)
    
    # 3. 修辞检测（比喻、排比等）
    rhetoric_patterns = ['像', '如', '仿佛', '如同']
    has_rhetoric = any(p in text for p in rhetoric_patterns)
    
    return diversity * 0.5 + sentence_variety * 0.35 + rhetoric_score
```

### 3.5 可读性评估

```python
def _evaluate_readability(self, text: str) -> float:
    """评估文本可读性"""
    score = 0.5  # 基础分
    
    # 1. 段落结构（合理分段加分）
    if has_reasonable_paragraphs(text):
        score += 0.15
    
    # 2. 标点密度（合理区间加分）
    punct_ratio = calculate_punctuation_ratio(text)
    if 0.04 <= punct_ratio <= 0.08:
        score += 0.2
    
    # 3. 对话标记（有对话加分）
    if has_dialogue_markers(text):
        score += 0.1
    
    # 4. 句长适中
    if average_sentence_length < 50:
        score += 0.05
    
    return min(score, 1.0)
```

### 5.2 API调用

```python
def _call_llm_api(self, prompt, max_tokens, temperature):
    """调用大模型API（含重试机制）"""
    for attempt in range(self.max_retries):
        try:
            response = self._api_client.chat.completions.create(
                model=self.default_model,
                messages=[
                    {"role": "system", "content": SYSTEM_PROMPT},
                    {"role": "user", "content": prompt}
                ],
                temperature=temperature,
                max_tokens=max_tokens,
                timeout=120
            )
            return response.choices[0].message.content
        except Exception as e:
            if attempt == self.max_retries - 1:
                raise
            time.sleep(1 * (attempt + 1))
```

## 六、使用示例

### 6.1 基本续写

```python
from plugins.continuation_generator_v1.plugin import ContinuationGeneratorPlugin
from core.models import ContinuationRequest

# 创建插件实例
plugin = ContinuationGeneratorPlugin()
plugin.set_api_client(openai_client)

# 构建请求
request = ContinuationRequest(
    starting_text="月光洒在窗台上，她轻轻叹了口气。",
    word_count=500,
    direction="natural",
    outline="女主角独自在家等待消息",
    characters=[
        {"name": "林婉儿", "role": "女主角", "traits": ["温柔", "坚强"]}
    ],
    worldview="现代都市背景",
    temperature=0.8
)

# 执行续写
result = plugin.generate_continuation(request)
print(result.text)
```

### 6.2 多版本生成并选择

```python
# 一站式生成并选择最佳版本
full_result = plugin.generate_and_select(
    request,
    num_versions=3,
    criteria={
        "coherence": 0.30,
        "style_match": 0.25,
        "word_count": 0.20,
        "creativity": 0.15,
        "readability": 0.10

)

print(f"最佳版本: {full_result['best_index'] + 1}")
print(f"最佳内容: {full_result['best_version'].text}")

```

### 6.3 流式生成
