## 三、核心回调方法

| 方法 | 功能 | 对应模块 |
|------|------|----------|
| `_on_refresh_memory_status()` | 刷新记忆层级状态 | 全局 |
| `_on_view_session_state()` | 查看当前会话状态 | L1热记忆 |
| `_on_clear_session_state()` | 清空会话状态 | L1热记忆 |
| `_on_save_session_snapshot()` | 保存会话快照 | L1热记忆 |
| `_on_rebuild_vector_index()` | 重建向量索引 | L2温记忆 |
| `_on_export_vector_data()` | 导出向量数据 | L2温记忆 |
| `_on_clear_vector_store()` | 清空向量库 | L2温记忆 |
| `_on_view_git_notes()` | 查看历史决策 | L3冷记忆 |
| `_on_add_git_note()` | 添加决策记录 | L3冷记忆 |
| `_on_edit_memory_md()` | 编辑MEMORY.md | L4精选档案 |
| `_on_daily_meditation()` | 每日冥想 | L4精选档案 |
| `_on_weekly_meditation()` | 每周大冥想 | L4精选档案 |
| `_on_archive_old_diary()` | 归档旧日记 | L4精选档案 |
| `_on_backup_memory()` | 一键备份记忆 | 维护工具 |
| `_on_restore_memory()` | 恢复记忆备份 | 维护工具 |
| `_on_clear_all_memory()` | 清空所有记忆 | 维护工具 |

## 五、使用示例

### 5.1 查看会话状态
```
设置 → 长期记忆管理 → L1热记忆 → 查看当前会话
```
弹出窗口显示JSON格式的会话状态，包括：
- 活跃任务
- 临时上下文
- 异常状态
- 待持久化数据

### 5.2 每日冥想
```
设置 → 长期记忆管理 → L4精选档案 → 每日冥想
```
自动执行：
1. 读取当日所有日记文件
2. 提取重要事件（high/medium重要性）
3. 判断是否值得保留（6个月后还有用吗？）
4. 更新MEMORY.md的工作记录部分

### 5.3 一键备份记忆
```
设置 → 长期记忆管理 → 维护工具 → 一键备份记忆
```
备份内容包括：
- .workbuddy/memory/ 目录
- data/vector_store/ 向量数据库
