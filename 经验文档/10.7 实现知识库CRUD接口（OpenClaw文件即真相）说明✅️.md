## 一、概述

### 1.1 设计目标

实现知识库CRUD接口，支持：
- 知识点的创建、读取、更新、删除
- 批量导入JSON文件
- 自动生成向量嵌入
- 分类知识库管理（科幻/玄幻/历史/通用）

### 1.2 技术选型

| 决策项 | 选择 | 说明 |

| 存储格式 | JSON | OpenClaw文件即真相原则 |
| 验证框架 | Pydantic | Schema验证 |
| 向量嵌入 | OpenAI / 本地模型 | 自动生成 |
| 单例模式 | 线程安全 | RLock保护 |

## 二、数据模型

### 2.1 KnowledgePoint（知识点）

```python
class KnowledgePoint(BaseModel):
    """知识点数据模型 - 符合Schema定义"""
    
    # 必填字段
    knowledge_id: str        # 知识点唯一标识（如scifi-physics-001）
    category: str            # 分类（scifi/xuanhuan/history/general）
    domain: str              # 知识领域（physics/chemistry/religion等）
    title: str               # 知识点标题
    content: str             # 知识点详细内容（50-5000字符）
    keywords: List[str]      # 关键词列表
    created_at: str          # 创建时间
    updated_at: str          # 更新时间
    
    # 可选字段
    embedding: Optional[List[float]]  # 向量嵌入
    references: Optional[List[str]]   # 参考来源
    difficulty: Optional[str]         # 知识难度（basic/intermediate/advanced）
    tags: Optional[List[str]]         # 自定义标签
    metadata: Optional[Dict[str, Any]] # 元数据
```

### 2.2 验证规则

| 字段 | 验证规则 |
|------|----------|
| `knowledge_id` | 格式：`{category}-{domain}-{序号}`，如scifi-physics-001 |
| `category` | 枚举：scifi/xuanhuan/history/general |
| `content` | 长度：50-5000字符 |
| `keywords` | 数量：2-10个，每个1-20字符 |

### 2.3 结果模型

```python
class KnowledgeCreateResult(BaseModel):
    """创建知识点结果"""
    success: bool
    knowledge_id: Optional[str]
    knowledge: Optional[KnowledgePoint]

class KnowledgeSearchResult(BaseModel):
    """检索结果"""
    knowledge_id: str
    title: str
    content: str
    category: str
    domain: str
    score: float
    keywords: List[str]

class ImportResult(BaseModel):
    """批量导入结果"""
    total: int
    success: int
    failed: int
    errors: List[Dict[str, str]]
    knowledge_ids: List[str]
```

### 5.5 相关性计算

```python
def _calculate_relevance(self, query: str, kp: KnowledgePoint) -> float:
    """计算相关性分数"""
    score = 0.0
    
    # 标题匹配
    if query in kp.title.lower():
        score += 0.5
    
    # 关键词匹配
    for keyword in kp.keywords:
        if query in keyword.lower():
            score += 0.3
    
    # 内容匹配
    if query in kp.content.lower():
        score += 0.2
    
    return min(score, 1.0)
```

## 六、使用示例

### 6.1 创建知识点

```python
from core import get_knowledge_manager

# 获取管理器
manager = get_knowledge_manager(workspace_root=Path("E:/project"))

# 创建知识点
result = manager.create_knowledge(
    category="scifi",
    domain="physics",
    title="时间膨胀效应",
    content="根据狭义相对论，当物体接近光速运动时，时间流速会变慢...",
    keywords=["相对论", "时间", "光速", "时间膨胀"],
    references=["《三体》", "《星际穿越》"],
    difficulty="intermediate",
    tags=["硬科幻", "物理学"]
)

if result.success:
    print(f"创建成功: {result.knowledge_id}")
else:
    print(f"创建失败: {result.error}")
```
