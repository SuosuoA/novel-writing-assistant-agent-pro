## 二、核心功能

### 2.1 九维度智能评分

| 维度 | 权重 | 检查内容 |
|------|------|----------|
| 世界观 | 12% | 世界观一致性检查 |
| 人设 | 19% | 人物性格、对话一致性 |
| 大纲 | 13% | 情节推进符合大纲 |
| 风格 | 19% | 写作风格匹配度 |
| 知识库 | 8% | 知识点引用质量 |
| 写作技巧 | 8% | 写作技巧应用 |
| 字数 | 8% | 字数达标率 |
| 上下文衔接 | 8% | 与前文衔接自然度 |
| AI感 | 5% | 文本自然度（越低越好） |

**权重总和**: 100%

### 2.2 强制检查机制

**【本章完】标记一票否决制**:
- 检查位置：章节结尾最后100字符
- 匹配模式：`【本章完】`、`[本章完]`、`（本章完）`、`(本章完)`、`本章完`
- 失败处理：总分强制为0，触发重新生成
- 自动补充：检测缺失时自动添加标记

### 2.3 优化建议生成

每个维度提供切实可执行的优化建议：
- **低分维度**（<0.5）：详细修改指南 + 具体示例
- **中分维度**（<0.7）：针对性优化建议
- **高分维度**（≥0.8）：保持当前表现

### 2.4 Claw记忆集成

**四层记忆架构**:
- L1热记忆：内存缓存（当前会话）
- L2温记忆：文件存储（持久化）
- L3冷记忆：反馈分析（学习模式）
- L4档案：长期优化趋势

**核心功能**:
- 存储评分结果和优化建议
- 检索历史相似问题
- 学习用户反馈模式
- 生成优化策略

### 2.5 本地模型辅助

**核心功能**:
- 语义相似度计算
- AI生成特征检测
- 降级方案（Jaccard相似度）

**模型路径**: `sentence_transformers_cache/`


## 五、使用方式

### 5.1 基本调用

```python
import sys
import importlib.util

# 动态加载模块
plugin_dir = r'plugins\expert-novel-v1'
spec = importlib.util.spec_from_file_location("models", f"{plugin_dir}/models.py")
models = importlib.util.module_from_spec(spec)
spec.loader.exec_module(models)

spec = importlib.util.spec_from_file_location("validator", f"{plugin_dir}/validator.py")
validator_module = importlib.util.module_from_spec(spec)
sys.modules['models'] = models
spec.loader.exec_module(validator_module)

# 创建验证器并评分
validator = validator_module.ExpertValidator()
evaluation = validator.evaluate(content, context)
```

## 六、配置说明

### 6.1 插件配置（config.yaml）

```yaml
expert:
  quality:
    threshold: 0.8          # 质量阈值
    max_iterations: 5       # 最大迭代次数
  
  mandatory_checks:
    chapter_end_marker:
      enabled: true         # 启用强制检查
      check_range: 100      # 检查最后100字符
  
  memory:
    enabled: true           # 启用记忆系统
    l1_ttl: 3600           # L1缓存时间
  
  local_model:
    enabled: true           # 启用本地模型
    model_name: "paraphrase-multilingual-MiniLM-L12-v2"
```

### 6.2 权重配置（config/expert_weights.yaml）

## 七、降级方案

### 7.1 降级路径

```
专家模式
    ↓ (加载失败)

    ↓ (加载失败)
缓存生成器
    ↓ (加载失败)

```

### 7.2 各模块降级策略

| 模块 | 失败场景 | 降级方案 |
|------|----------|----------|
| ExpertPlugin | 插件加载失败 | 使用novel-generator-v3 |
| ExpertValidator | 评分失败 | 使用基础评分（各维度0.5） |
| LocalModelAssistant | 模型加载失败 | 使用Jaccard相似度 |
| ExpertMemory | 存储失败 | 禁用记忆，仅用缓存 |

```python
def _on_expert_changed(self, expert_info):
    """专家选择变化回调"""
    if expert_info:
        self._set_status(f"已选择专家模式：{expert_info.name}")
    else:
        self._set_status("已选择默认模式")

def _get_expert_config(self) -> dict:
    """获取当前专家配置"""
    if self._expert_selector:
        return self._expert_selector.get_config()
    return {}

def _is_expert_mode_enabled(self) -> bool:
    """检查是否启用了专家模式"""
    return self._expert_selector is not None and self._expert_selector.get_selected_expert() is not None
```
